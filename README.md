# 病院防災フレームワーク（8軸+α）

## プロジェクト概要

本プロジェクトは、災害時のインフラ途絶に対応するための病院防災マニュアル作成を支援するものです。

**27施設のアンケート結果**に基づき、現場の声を反映した実践的なフレームワークを提供します。

### フレームワーク構成

病院のライフラインを体系的に整理し、各インフラが途絶した際の対応手順を標準化するフレームワークです。

| 軸 | カテゴリ | 主な検討事項 |
|---|---------|------------|
| **共通** | アクションカード・災害BOX | 30秒で動ける行動指針、標準備品セット |
| 1 | 電気 | 非常用電源、優先供給エリア、燃料確保、**ブラックアウト対策** |
| 2 | 水道 | 受水槽容量、節水対策、代替水源 |
| 3 | 医療ガス | 酸素ボンベ備蓄、配管系統、使用制限 |
| 4 | 通信 | 衛星電話、無線機、**LINE連絡網、トランシーバー運用** |
| 5 | 情報システム | 電子カルテ停止時対応、紙運用への切替 |
| 6 | エレベーター | 患者搬送代替ルート、階段搬送体制 |
| 7 | 空調 | 温度管理優先エリア、代替冷暖房 |
| 8 | 給食・廃棄物・燃料 | 備蓄食、廃棄物保管、燃料調達、**離島長期備蓄** |
| **9** | 組織体制 | 発災スイッチ、指揮命令系統、多職種連携 |

### 地域別の特性

| 地域 | キーワード | 重点課題 |
|-----|----------|---------|
| 離島 | 津波、備蓄、避難 | 「生き延びる」- 救援到着までの自己完結 |
| 都市部 | 通信、参集、トリアージ | 「さばく」- 大量傷病者と情報の処理 |
| 寒冷地 | 電源、暖房、燃料 | 「凍えない」- ブラックアウト対策 |

## 活用方法

1. **自施設の現状把握**: 8軸フレームワークに沿って自施設の脆弱性を確認
2. **アクションカード作成**: テンプレートを活用して各インフラ途絶時の行動手順を作成
3. **訓練・演習**: 作成したアクションカードを用いた実地訓練
