# 「蛇口をひねっても水が出ない」だけが断水ではない

**〜下水ポンプの逆流、自動水栓の罠、揚水ポンプの盲点——27施設が見落とした「水の3つの死角」〜**

---

## はじめに：水は「見える生命線」——だからこそ盲点がある

電気は見えません。医療ガスも見えません。だから停電やガス供給停止は「気づきにくい」リスクとして認識されている。

では、水はどうか。

水は見えます。蛇口をひねれば出る。止まれば分かる。だから「断水対応」は比較的シンプルだと思われている。

**その認識が、最大の盲点だ。**

27施設のアンケートを分析した結果、水に関する脆弱性は「蛇口から水が出ない」という単純な断水だけではありませんだった。**3つの死角**が、多くの病院で見落とされていた。

1. **下水ポンプ停止による逆流** ——水が来ないのではなく、水が「出せない」
2. **自動水栓の罠** ——蛇口はあるのに、手が洗えない
3. **揚水ポンプの盲点** ——受水槽に水があるのに、上階は断水

この3つは、いずれも**停電との複合災害**で発生します。そして、多くの施設で「考えたことがなかった」と回答された項目だ。

---

## 第1章：下水ポンプ停止——「水が出せない」という断水

### 見落とされた排水系統

断水対策を聞くと、多くの施設が「受水槽の残量」「給水車の手配先」「ペットボトルの備蓄」を挙げます。これらはすべて**給水側**の対策だ。

しかし、水には「入ってくる」だけでなく「出ていく」もあります。**排水系統**だ。

地下に排水ポンプを持つ施設——つまり、下水本管より低い位置に建物の一部がある施設——では、排水を電動ポンプで汲み上げて下水に流している。

**停電すると、このポンプが止まります。**

### 逆流のメカニズム

```
【通常時】
  トイレ・洗面 → 排水管 → 地下排水槽 → 排水ポンプ → 下水本管
                                          ↑ 電力で稼働

【停電時】
  トイレ・洗面 → 排水管 → 地下排水槽（満水） ← 下水が逆流
                                                ↓
                                          地下設備室に浸水
```

逆流した下水は、地下の設備室、電気室、サーバー室を浸水させます。これらの部屋が汚水で浸水すれば、**電力復旧後もシステム復旧ができない**という二次災害に発展する。

### 2018年北海道ブラックアウトの教訓

2018年の北海道胆振東部地震では、道内全域が停電（ブラックアウト）しました。複数の病院で排水ポンプが停止し、地下への逆流リスクが現実のものとなりた。

ある施設では、停電開始から数時間で地下排水槽が満水に近づき、急遽トイレの使用を全面停止。簡易トイレへの切替を余儀なくされた。

**問題は、この切替判断の基準がどの施設にも存在しなかったことだ。**

## 第2章：自動水栓の罠——蛇口があるのに手が洗えない

### 感染管理の進化が生んだ皮肉

現代の病院は感染管理の観点から、手洗い場にタッチレス（自動センサー式）水栓を広く導入している。手を触れずに水が出る。感染リスクを下げる合理的な設計だ。

しかし、この合理的な設計が、**停電時に完全に裏目に出る。**

自動水栓はセンサーと電磁弁で制御されている。停電すると：

- センサーが動かない → 手を近づけても反応しない
- 電磁弁が閉じたまま → 物理的に水が出ない
- 手動ハンドルがない → 蛇口を「ひねる」ことができない

**水道管に水は来ている。受水槽にも水はある。なのに、手が洗えない。**

### 感染管理の崩壊

手洗いは感染管理の基本中の基本です。停電時に手が洗えなければ：

- 手術前の手洗いができない
- 処置前後の手指衛生ができない
- 食事配膳時の衛生が保てない

アルコール消毒で代替できる場面もありますが、目に見える汚れがある場合や、芽胞形成菌（クロストリジウム・ディフィシル等）への対応では、流水手洗いが要る。

### 27施設の現実

アンケートで「自動水栓の停電時対応」を確認したところ：

- 自動水栓の設置箇所を把握している：**4施設**
- 停電時に手動で使える水栓の場所を周知している：**2施設**
- 自動水栓の停電対応マニュアルがある：**0施設**

**ゼロだ。**

## 第3章：揚水ポンプの盲点——受水槽に水があるのに上階は断水

### 水は上に流れない

当たり前のことですが、水は重力に従います。受水槽（通常は地下または1階に設置）から上階に水を送るには、揚水ポンプが要る。

このポンプが停電で停止すると、**上階には水が届きません。**

「受水槽に水が3日分ある」は、「3日間水が使える」とは限りません。揚水ポンプが非常用電源に接続されていなければ、受水槽の水は地下に眠ったまま、上階の病棟は断水する。

### 高架水槽がある施設の落とし穴

一部の施設は高架水槽（屋上の水槽）を持っています。「高架水槽があるから、停電でも重力で水が出る」——これは正しい。

しかし、高架水槽の容量は限られています。通常の使用量で**数時間〜半日**で枯渇します。枯渇後は揚水ポンプが動かない限り補充できない。

### 確認すべき3つの問い

1. **揚水ポンプは非常用電源に接続されているか？**
2. **接続されていない場合、上階への手動給水手段はあるか？**（バケツリレー、ポンプ車等）
3. **高架水槽がある場合、枯渇までの時間を計算しているか？**

27施設のうち、揚水ポンプの非常電源接続状況を**即答できた**施設は5施設でした。「施設課に聞かないと分からない」が大半の回答だ。

## 第4章：「排水BCP」という発想

### 給水BCPだけでは足りない

従来の断水対策は「給水BCP」でした。水をどう確保するか。受水槽、給水車、ペットボトル、井戸。すべて「水を入れる」側の計画だ。

しかし本記事で見てきたように、**「水を出す」側にも等しく計画が必要**だ。

私はこれを**「排水BCP」**と呼びます。

### 排水BCPの4つの柱

| 柱 | 内容 | 確認事項 |
|----|------|---------|
| 排水ポンプ電源 | 排水ポンプの非常電源接続 | 電気系統図で確認 |
| 逆流防止 | 逆流防止弁の設置・点検 | 実地確認・定期点検 |
| 浸水対策 | 地下設備室の防水 | 止水板・排水溝の確認 |
| 使用停止判断 | トイレ使用停止と簡易トイレ切替の基準 | 排水槽満水時の手順 |

### なぜ排水BCPが見落とされるのか

理由は単純です。**排水は「見えない」から**だ。

蛇口をひねって水が出なければ、誰でも断水に気づくる。しかし、排水が止まっていることには、**地下が浸水するまで気づきません。**

「見える断水」には対策がある。「見えない断水」には対策がない。これが27施設の現実だった。

## 第5章：27施設のアンケートから見えた「水」の脆弱性

### 数字が語る現実

| 項目 | 対応済み | 未確認 |
|------|---------|-------|
| 受水槽残量の把握 | 19施設 | 8施設 |
| 給水車受入手順 | 12施設 | 15施設 |
| 揚水ポンプの非常電源確認 | 5施設 | 22施設 |
| 排水ポンプの非常電源確認 | 3施設 | 24施設 |
| 自動水栓の停電対応 | 0施設 | 27施設 |
| 簡易トイレ切替基準 | 7施設 | 20施設 |

**下に行くほど「見えない」項目であり、対策率が下がる。** これは偶然ではない。

### 3つの施設群

アンケート結果から、水の対策レベルは3つの段階に分類されます。

| レベル | 施設数 | 特徴 |
|-------|-------|------|
| 給水対策済み | 12施設 | 受水槽・備蓄水・給水車の手配はできている |
| 給水+節水計画あり | 7施設 | 節水プロトコル・優先用途の決定まで完了 |
| 排水BCPまで対応 | 0施設 | 排水系統・自動水栓・揚水ポンプまで対策済み |

**排水BCPに到達している施設は、27施設中ゼロだった。**

## 第6章：排水制限の院内放送テンプレート

### 「今すぐ水を流さないで！」をどう伝えるか

下水ポンプが停止し、排水制限が必要になったとき、最大の課題は**「全館への周知」**だ。

院内放送が使える場合はまだいい。しかし停電で院内放送が使えない場合は、伝令・メガホン・LINEの併用が必要になります。いずれの手段であっても、**伝える内容は統一されていなければならない。**

以下は、排水制限を院内に伝えるための放送テンプレートです。事前に印刷し、災害BOXとナースステーションに配備すべきだ。

### 放送文テンプレート（レベル別）

#### レベル1：排水制限（トイレ以外の排水を停止）

```
【院内放送（排水制限）】

こちらは災害対策本部です。
現在、停電により排水ポンプが停止している。

ただいまより、排水制限を発令する。

・トイレ以外で水を流さないでください。
・シャワー、入浴、洗濯は中止です。
・手洗いはアルコール消毒に切り替えてください。
・氷や飲み残しをシンクに捨てないでください。

トイレは引き続き使用可能ですが、
節水にご協力ください。

繰り返します。トイレ以外で水を流さないでください。
ご協力をお願いする。
```

#### レベル2：全面排水停止（トイレを含む全排水を停止）

```
【院内放送（全面排水停止）— 緊急】

こちらは災害対策本部です。緊急のお知らせだ。

ただいまより、すべての排水を停止する。

・トイレの水を絶対に流さないでください。
・蛇口から水を出さないでください。
・シンク、洗面台、すべての排水口を使用禁止とする。

排水管から下水が逆流する危険がある。
水を流すと、地下の設備室が浸水し、
病院全体の復旧がさらに遅れます。

トイレは簡易トイレに切り替えます。
簡易トイレは [場所] に設置する。

繰り返する。
すべての排水を停止すべきだ。
水を流さないでください。
ご協力をお願いする。
```

### 放送文の設計思想

| 要素 | 理由 |
|------|------|
| 「災害対策本部です」で始める | 権限ある指示であることを明示 |
| 禁止事項を具体的に列挙 | 「節水」では曖昧。「流すな」が正しい |
| 理由を簡潔に説明 | 「逆流する」「復旧が遅れる」で協力を引き出す |
| 代替手段を提示 | 「アルコール消毒」「簡易トイレ」で不安を減らす |
| 繰り返す | パニック時は一度では伝わらない |

### 院内放送が使えない場合

| 代替手段 | 使い方 |
|---------|-------|
| メガホン | 各フロアを巡回し、レベル2の放送文を読み上げ |
| 伝令 | 各病棟に放送文の紙を配布。師長が病棟内に周知 |
| LINE | 放送文をテキストで一斉送信。写真で掲示も共有 |
| 掲示 | トイレ入口・洗面台・シンクに「使用禁止」の紙を貼る |

> **最も効果的な手段**: トイレの個室ドア内側に「使用禁止」の紙を貼ること。目に入らない人はいない。

## おわりに：見える断水と見えない断水

「蛇口をひねっても水が出ない」は、見える断水です。対策もしやすい。

しかし本当に怖いのは、**見えない断水**だ。

- 下水が逆流して地下設備室が浸水する
- 自動水栓が動かず手が洗えない
- 受水槽に水があるのに上階に届かない

これらはすべて、**停電との複合災害**で発生します。そして、ほとんどの施設で「考えたことがなかった」項目だ。

今日できることは3つだ。

1. **揚水ポンプの電源を確認する**（非常用電源に接続されているか）
2. **排水ポンプの電源を確認する**（逆流防止弁はあるか）
3. **自動水栓の場所をリストアップする**（停電時に使える手動水栓はどこか）

この3つの確認に、特別な予算も設備も必要ない。必要なのは、「見えない断水」の存在を知ることだけだ。
