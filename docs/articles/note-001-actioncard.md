# 「30秒で動ける」アクションカードの作り方

**〜27施設のアンケートから見えた、現場が本当に求めているもの〜**

---

## はじめに

「マニュアルは分厚いけど、いざという時に読めない」

これは、全国27施設の病院防災アンケートで最も多く聞かれた声だった。

災害は突然やってくる。停電、断水、通信途絶...。その時、分厚いマニュアルを開いている余裕はない。

**必要なのは「30秒で理解でき、即座に行動に移せる」ツール。**

それが**アクションカード**だ。

---

## アンケートで見えた「現場の本音」

27施設へのアンケートで、「自施設で取り組みたいこと」の第1位は断トツで**「アクションカード・マニュアルの整備」**だった。

現場から挙がった具体的な声：

> 「全スタッフが理解できるシンプルなマニュアルへの改訂が必要」
>
> 「部署ごとの役割分担の明確化」
>
> 「アクションカードと災害BOXをセットにしたい」

つまり、現場が求めているのは：

1. **シンプルであること**（読む時間がない）
2. **役割が明確であること**（誰が何をするか迷わない）
3. **すぐ手に届くこと**（探す時間がない）

---

## アクションカードの3原則

### 原則1: 30秒で理解できる

**NG例:**
```
停電発生時は、まず落ち着いて状況を確認し、
非常用発電機の稼働状況を施設課に確認すべきだ。
その後、各部署のリーダーは患者の安全確認を行い...
（以下200行続く）
```

**OK例:**
```
【停電発生】
  ↓
患者の安全確認（30秒）
  ↓
本部へ報告（内線○○○）
  ↓
指示を待つ
```

フロー図で、**視線の流れ**だけで次のアクションが分かるようにする。

### 原則2: 判断を迷わせない

「状況に応じて適切に対応」という表現は禁止だ。

**NG例:**
```
状況に応じて、必要であれば応援を要請すべきだ。
```

**OK例:**
```
患者に危険あり？
  │
  ├─ YES → 応援要請（内線○○○）
  │
  └─ NO → 状況報告のみ
```

Yes/Noで分岐させ、**誰がやっても同じ判断**になるようにする。

### 原則3: その場で完結する

「詳細は○○マニュアル参照」は禁止だ。

カード内に必要な情報を全て記載します：
- 連絡先（内線番号、携帯番号）
- 物品の場所（災害BOXの位置）
- 次のアクション

## アクションカードの基本フォーマット

### 表面（A4サイズ）

```
┌──────────────────────────────────────────────┐
│ 【役割】病棟リーダー                          │
│ 【事象】停電発生時                            │
├──────────────────────────────────────────────┤
│                                              │
│         ┌───────┐                           │
│         │ 停電発生 │                          │
│         └────┬────┘                          │
│              ▼                               │
│         ┌─────────────┐                      │
│         │ 患者安全確認 │                      │
│         │  （30秒）   │                      │
│         └──────┬──────┘                      │
│                ▼                             │
│    ┌───────────────────────┐                 │
│    │  患者に危険はあるか？  │                 │
│    └───────────┬───────────┘                 │
│          │           │                      │
│         YES          NO                      │
│          ▼           ▼                       │
│    ┌─────────┐ ┌─────────┐                  │
│    │応援要請 │ │状況報告 │                   │
│    │内線○○○│ │内線○○○│                   │
│    └─────────┘ └─────────┘                  │
│                                              │
├──────────────────────────────────────────────┤

```

### 裏面

- 詳細手順（箇条書き5項目以内）
- 関連するカード番号
- 更新日・作成者

## 災害BOXとセットで配備する

アクションカードは、**災害BOX**と一緒に配備してこそ意味がある。

### 災害BOX標準セット

| カテゴリ | 品目 |
|---------|-----|
| 情報 | アクションカード一式、緊急連絡先リスト |
| 通信 | トランシーバー、予備電池 |
| 照明 | LEDランタン、懐中電灯 |
| 記録 | 筆記用具、クロノロジー用紙 |
| 安全 | 軍手、ヘルメット |

設置場所は**全スタッフが知っている場所**に。
「あそこにある」と誰でも言えることが大切だ。

## 発災スイッチも明確に

アンケートでもう一つ多かった声が：

> 「災害のスイッチ（発災宣言）の入れ方・基準の明確化」
> 「夜間・休日は誰が判断するのか」

### 発災スイッチの基準例

| レベル | 基準 | 宣言者 |
|-------|-----|-------|
| レベル1 | 震度4以上 | 当直責任者 |
| レベル2 | 震度5弱以上、インフラ1つ途絶 | 当直責任者 |
| レベル3 | 震度5強以上、インフラ2つ以上途絶 | **誰でも可** |

**重要:** 迷ったら「上のレベル」で宣言する。
空振りは許される。見逃しは許されない。

---

## まずは1枚から始めよう

完璧なアクションカードを全部署分作ろうとすると、永遠に完成しない。

### ステップ1: 最も不安なシナリオを1つ選ぶ
例：夜間の停電

### ステップ2: そのシナリオで最初にやることを3つ書く
1. 患者安全確認
2. 本部報告
3. 指示待ち

### ステップ3: フロー図にする
紙とペンで十分だ。

### ステップ4: 実際に使ってみる
訓練で使い、改善する。

**「とりあえず1枚」が、100ページのマニュアルより価値があります。**

---

## おわりに

災害は、準備している人のところにも、していない人のところにも、平等にやってくる。

違いが出るのは、**その瞬間に動けるかどうか**。

30秒で理解でき、迷わず動ける。
そんなアクションカードを、ぜひ自施設で作ってみろ。

次回は「通信途絶でも繋がる！LINE・トランシーバー運用術」をお届けする。
