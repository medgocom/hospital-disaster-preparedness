# あなたの病院の「見えない生命線」は、停電で静かに死ぬ

**〜医療ガス監視盤の電源、開かない窓、冷蔵庫の中の薬——27施設が見落とした「空気と温度の盲点」〜**

---

## はじめに：見えないものは、止まっても気づかない

酸素は見えません。空気も見えません。温度も見えない。

**だから、止まっても誰も気づかない。**

午前3時、停電。非常灯が点くる。非常用電源が起動し、人工呼吸器が動いていることを確認します。電子カルテは落ちましたが、紙運用に切り替えます。

ここまでは、多くの施設が訓練している。

しかし、その裏で**3つの「見えない死」**が静かに進行している。

1. **医療ガスの監視盤が暗転**——酸素残量もアラームも消えた。異常に気づくのは、患者のSpO2が下がってから。
2. **空調が停止し、密閉された建物の温度が上昇**——窓が開かない。CO2が溜まる。サーバー室が過熱する。
3. **薬品冷蔵庫の温度が上がり始める**——インスリン、ワクチン、血液製剤。使えなくなるまでのカウントダウンが始まった。

27施設のアンケートで、これらの「見えない生命線」の対策状況を確認しました。結果は、驚くべきものだった。

---

## 第1章：監視盤の電源という盲点

### 見えない酸素を見るための装置が、停電で消える

医療ガスの監視盤（セントラルモニター）は、酸素・笑気・圧縮空気・吸引の供給圧力と残量を一元的に監視する装置です。配管の異常、残量低下、系統間の切替をアラームで知らせます。

**この監視盤が、非常用電源に接続されていない施設がある。**

非常用電源に接続されていなければ、停電と同時に監視盤は暗転します。すると：

- ガス残量の把握ができない
- 配管異常のアラームが鳴らない
- CE（液化酸素タンク）とマニフォールド（ボンベ）の自動切替が停止する

### 「人工呼吸器は動いているから大丈夫」の落とし穴

人工呼吸器は非常用電源に接続されている施設がほとんどです。だから停電直後は動いている。

しかし、**酸素の供給元が尽きれば、呼吸器が動いていても意味がない。**

監視盤が停止している状態では、酸素の残量が見えません。CEからマニフォールドへの自動切替も動きません。気づいたときには酸素圧が低下し、呼吸器がアラームを出す——その時点で、すでに患者は低酸素状態だ。

### 27施設のデータ

| 項目 | 施設数 |
|------|-------|
| 監視盤が非常用電源に接続されている | 8施設 |
| 接続されていない | 7施設 |
| 把握していない | 12施設 |

**半数近くの施設が「把握していない」。** これが最も危険な回答だ。

---

## 第2章：開かない窓——密閉建築の罠

### なぜ現代の病院は窓が開かないのか

近年の病院建築は、感染管理・省エネルギー・防音の観点から、**窓を開けられない設計（sealed building）**が増えています。空調が室内環境の全てを制御する前提の設計だ。

この設計は、空調が動いている限り合理的だ。

**空調が止まった瞬間、合理性は崩壊します。**

### 密閉建築で起こること

空調停止後の時間経過で、密閉建築では以下のリスクが段階的に発生する。

| 経過時間 | リスク | 影響 |
|---------|-------|------|
| 30分〜 | CO2濃度上昇 | 頭痛、判断力低下 |
| 1〜2時間 | 室温上昇（夏季） | 熱中症リスク。サーバー室の過熱開始 |
| 2〜4時間 | 室温上昇（夏季続き） | 薬品温度逸脱。患者の体調悪化 |
| 同時 | 手術室陽圧消失 | 感染リスク上昇。進行中手術の判断が必要 |
| 同時 | 陰圧病室の機能喪失 | 感染症患者からの空気感染リスク |

### 窓が1枚も開かないフロア

27施設に「各フロアで開けられる窓はあるか」を確認したところ：

- 全フロアで開けられる窓がある：**6施設**
- 一部のフロアで開けられない：**14施設**
- 窓の開閉状況を把握していない：**7施設**

**「窓が開かないことをリスクとして認識していた」施設は3施設のみ。**

---

## 第3章：冷蔵庫が止まった瞬間

### 最初に死ぬのは、患者ではなく薬

停電で冷蔵庫が止まると、温度が上がり始めます。開けなければ数時間は保冷されますが、頻繁に開閉すれば急速に温度が上昇する。

温度逸脱が致命的な物品：

| 物品 | 保管温度 | 逸脱許容時間 | 逸脱後の影響 |
|-----|---------|------------|-------------|
| インスリン | 2〜8℃ | 数時間（条件による） | 効力低下・使用不可 |
| ワクチン | 2〜8℃ | 数時間 | 失活・廃棄 |
| 血液製剤 | 製剤ごと | 30分〜数時間 | 使用不可・廃棄 |
| 検査試薬 | 製品ごと | 数時間 | 精度低下 |

### 薬品冷蔵庫は非常用電源に接続されているか

多くの施設で、人工呼吸器やモニターは非常用電源に接続されている。しかし、**薬品冷蔵庫が非常用電源に接続されている施設は少数**だ。

27施設の結果：

| 項目 | 施設数 |
|------|-------|
| 薬品冷蔵庫が非常用電源に接続 | 5施設 |
| 接続されていない | 14施設 |
| 把握していない | 8施設 |

停電が24時間を超えた場合、非常電源に接続されていない施設では、**数百万円分の薬品が廃棄**になる可能性がある。

---

## 第4章：手術室の陽圧が消える時

### 見えないバリアの崩壊

手術室は陽圧（室内の気圧が廊下より高い状態）に維持されている。これにより、ドアを開けても廊下の空気が手術室に入り込まず、清浄度が保たれます。

空調停止 = 陽圧消失だ。

手術室のドアを開けた瞬間、廊下の空気が手術室に流入し、清浄度が崩壊する。

### 進行中手術の判断

空調停止時に手術が進行中の場合、以下の判断が要る。

| 手術状況 | 推奨判断 |
|---------|---------|
| 閉創間近 | 速やかに閉創・終了 |
| 術中（中盤） | 可能な限り早期に安全な段階まで進め終了 |
| 開始直後 | 中止・延期を検討 |
| 待機中 | 延期 |

**問題は、この判断基準が事前に定められていない施設が大半であること。** 27施設のうち、空調停止時の手術判断基準を持つ施設は4施設だった。

## 第5章：対策の設計思想——「見えない」を「見える」にする

### 原則：見えないリスクは、チェックリストで見えるようにする

見えない生命線への対策は、大規模設備投資だけではない。**「見えない」を「見える」に変える仕組み**が最も重要だ。

### 医療ガス：監視の代替

| 対策 | コスト | 効果 |
|------|-------|------|
| 監視盤を非常用電源に接続 | 中〜高 | 根本解決 |
| 手動残量確認手順の整備 | 低 | 代替手段 |
| 酸素ボンベ使用時間早見表の配備 | 低 | 判断支援 |
| 夜間ガス異常対応者の指定 | 低 | 体制整備 |

### 空調・換気：密閉からの脱出

| 対策 | コスト | 効果 |
|------|-------|------|
| 開けられる窓のフロアマップ作成 | 低 | 現状把握 |
| 窓開け・換気ルート図の作成 | 低 | 効率的換気 |
| ポータブル換気扇の備蓄 | 低〜中 | 応急換気 |
| 手術室運用判断基準の策定 | 低 | 判断支援 |
| サーバー室用ポータブル冷房 | 中 | IT保護 |

### 窓開け・換気ルート図の作り方

空調停止時に「窓を開ければいい」と分かっていても、**どの窓をどの順序で開ければ風が通るか**を知らなければ、効果的な換気はできない。以下の手順で「窓開け・換気ルート図」を作成すべきだ。

#### Step 1: 開閉調査（所要：各フロア30分）

各フロアを歩き、すべての窓について以下を記録する。

| 確認項目 | 記録内容 |
|---------|---------|
| 窓の位置 | 図面上にマーキング |
| 開閉可否 | ◯ 開く / × 開かない / △ 鍵が必要 |
| 開口方式 | 引き違い / 上げ下げ / 突き出し / FIX（固定） |
| 開口面積 | 大（1m²以上）/ 中 / 小（換気口程度） |
| 方角 | 北 / 南 / 東 / 西 |
| 障害物 | カーテン / 棚 / 医療機器が前にある |

#### Step 2: 風の道を設計する

効率的な換気には**「入口」と「出口」**が要る。同じ方角の窓を2つ開けても風は通りない。

```
【効果的な換気】
  南側の窓（風の入口）
    ↓ 廊下・病室を通過 ↓
  北側の窓（風の出口）
  → 風が抜ける

【効果のない換気】
  南側の窓A ←→ 南側の窓B
  → 同じ方角。風が通らない
```

**原則：対角線上の窓を開ける。** 南と北、東と西の窓をセットで開けることで、風の道がでくる。

#### Step 3: 換気ルート図の完成

フロア図面上に以下を記入する。

- **緑マーク**: 開けられる窓（開閉方式・開口面積を記載）
- **赤マーク**: 開けられない窓（FIX窓）
- **青矢印**: 推奨換気ルート（風の入口→出口）
- **黄マーク**: 鍵が必要な窓（鍵の保管場所を記載）

#### Step 4: 配備と訓練

- 完成した換気ルート図を各フロアのナースステーションに掲示
- 鍵付き窓の鍵を災害BOXに追加
- 停電訓練時に「窓開け」をチェック項目に含める

> **見落としがちなポイント**: エレベーターホール、階段室、廊下の窓は見落とされやすいが、換気ルートとして最も有効な場合が多い。

### 温度管理：コールドチェーンの死守

| 対策 | コスト | 効果 |
|------|-------|------|
| 薬品冷蔵庫を非常用電源に接続 | 中 | 根本解決 |
| 保冷剤・クーラーボックスの備蓄 | 低 | 応急保冷 |
| 温度逸脱時の廃棄判断基準 | 低 | 判断支援 |
| 「冷蔵庫を開けない」ルールの周知 | 低 | 保冷時間延長 |

---

## おわりに：見えない生命線を、見えるようにする

酸素。空気。温度。

この3つは、電気や水と違って**見えません。** 見えないから、止まっても気づかない。気づいたときには、患者のSpO2が下がり、薬品が使えなくなり、手術室の清浄度が崩壊している。

私たちにできることは、**見えないものを見えるようにすること**だ。

- 監視盤の電源を**確認する**（見えない酸素を見える化する装置の保護）
- 窓の開閉を**調査する**（見えない空気の逃げ道の確認）
- 冷蔵庫の電源を**確認する**（見えない温度変化の防止）

**見えない生命線は、確認しなければ存在しないのと同じだ。**

今日、施設課に電話すべきだ。「監視盤の電源、非常用に繋がっていますか？」と。

その一本の電話が、見えない生命線を守る第一歩になります。
