# 深夜2時、震度6。あなたは「発災宣言」を出せますか？

**〜「超軽量アクションカード」と「免責条項」で、夜間の初動凍結を終わらせる〜**

---

## はじめに：当直医の孤独

午前2時。強い揺れで目が覚めます。

あなたは当直医です。非常灯だけがぼんやり光る廊下。PHSは沈黙している。夜勤の看護師は3名、当直事務が1名。合計5名。

入院患者は150名。うち人工呼吸器装着が3名、透析中が1名。

院長の携帯は、出ない。

この瞬間、あなたはこの病院で最も重い判断を迫られている。**発災宣言を出すか、出さないか。**

そして、あなたの頭の中にはこんな声が聞こえている。

> 「本当に出していいのか？」
> 「大したことなかったら、責任を問われないか？」
> 「院長に確認してからのほうが安全では？」

**この3秒の迷いが、初動を殺します。**

前回の記事（[なぜ病院のマニュアルは「ゴミ」になるのか？](https://github.com/medgocom/hospital-disaster-preparedness/blob/main/docs/articles/note-003-real-bcp.md)）で指摘した4つの絶望。その根っこにあるのは、実は「権限」の問題ではなく、**「免責」の問題**だった。

---

## 第1章：なぜ当直医は「動けない」のか

### 権限委譲だけでは不十分だった

前回、私たちは「事前包括委任」を提案しました。当直医師に発災宣言の権限を事前に委任する仕組みだ。

しかし、27施設へのヒアリングを深掘りすると、問題の本質は「権限がない」ことではありませんだった。

> 「権限があっても、怖くて出せない」

これが本音だ。

### 恐怖の正体：「空振り」への罰

当直医が発災宣言をためらう理由を整理すると、3つの恐怖に集約されます。

| 恐怖 | 具体的な声 | 本質 |
|------|----------|------|
| 過剰対応への非難 | 「大げさだった」と言われる | 空振りが罰される文化 |
| 結果責任の追及 | 「あの判断は間違いだった」と事後検証で追及される | 善意の判断が保護されない |
| キャリアへの影響 | 「あの夜の対応が人事に響くのでは」 | 安全側に振れるインセンティブがない |

権限を渡しても、「使ったら罰される」と感じていれば、誰も使いない。

**権限委譲の次に必要なのは、免責だ。**

---

## 第2章：免責条項という発明

### 「善意の判断を組織が守る」

私が提案するのは、**災害時の免責条項**だ。

```
┌─────────────────────────────────────────────┐
│                                             │
│   【災害時免責条項】                         │
│                                             │
│   災害発生時またはその疑いがある状況で、      │
│   当直者が善意に基づき行った判断・行動は、    │
│   その結果が最善でなかった場合であっても、    │
│   懲戒処分および人事評価上の不利益の          │
│   対象としない。                             │
│                                             │
│   「空振り」は組織として正しい判断と          │
│   位置づけ、事後の非難を禁止する。           │
│                                             │
│                       病院長署名             │
│                                             │
└─────────────────────────────────────────────┘
```

### なぜ院長の署名が必要なのか

免責条項が「お気持ち表明」で終わらないために、**院長の署名を付す**ことが決定的に重要だ。

理由は3つ：

1. **法的な重み**: 署名があることで、病院としての公式な意思表示になる
2. **心理的な安全**: 「院長が許可した」という事実が、当直医の心理的ブレーキを外す
3. **事後検証での盾**: 万が一の事後検証でも「組織の方針として認められた行動だった」と言える

署名のない免責条項は、書かれていないのと同じだ。

### 「空振り」は正しい判断である

免責条項の核心は、**空振りの正当化**だ。

地震が来て、発災宣言を出して、結果的に大きな被害がなかった。これは「過剰対応」ではない。**情報が不十分な状況で、安全側に判断した正しい行動**だ。

航空業界では「Go-Around（着陸やり直し）」を判断したパイロットは称賛されます。実際には着陸できたかもしれない。でも、不安を感じたら迷わずやり直す。その判断を組織が守る。

病院の災害対応も同じだ。

> **「大したことなかったね」は、最高の結末。**

## 第3章：超軽量アクションカードに免責を焼き込む

### カードの冒頭が全てを決める

超軽量アクションカード v2.0は、最初に目に入る場所に免責条項を配置している。

従来のカード：
> 「停電発生時の対応フロー」から始まる

超軽量版 v2.0：
> 「あなたの権限と免責」から始まる

**手順の前に安心を。** これが設計思想だ。

災害は手順通りには進みません。しかし、「自分は守られている」という確信があれば、人は動けます。

### スマホ1画面の構成

カードは4つのセクションで構成されている。

| セクション | 内容 | スマホ表示 |
|-----------|------|----------|
| SECTION 0 | 権限と免責の宣言 | 最初のスクロールで必ず目に入る |
| SECTION 1 | 5項目チェック（チェックボックス付き） | 指で1つずつタップして確認 |
| SECTION 2 | 10分タイムラインのフロー図 | 今何分かで、次の行動が分かる |
| SECTION 3 | NG時の即時対応と「絶対やるな」リスト | 異常が出たとき、やるべきこと・やってはいけないことが一目で分かる |

### 5項目チェックの設計根拠

緊急時に人間が処理できる情報は5つまで（マジカルナンバー）。3系統のフレームワーク（電気・水道・医療ガス）+ 患者安全 + 通信から、各1項目ずつ抽出した。

| # | 確認項目 | なぜ最優先か |
|---|---------|------------|
| 1 | 重症患者は安全か | 呼吸器停止は数分で致命的 |
| 2 | 電気は来ているか | 非常灯1つで電気系統全体の生死が分かる |
| 3 | 水は出るか | 蛇口1つで断水の有無が判定できる |
| 4 | 酸素は来ているか | 配管停止は呼吸器患者に直結 |
| 5 | 外と連絡できるか | 「孤立」か否かの分水嶺 |

5つ全部OKなら、「この病院は今すぐ崩壊しない」と判断でくる。1つでもNGなら、どの系統がやられたか即座に分かり、次のアクションが決まります。

## 第4章：「やるな」リストの威力

### 災害時、最も危険なのは「善意の暴走」

超軽量カード v2.0には、「やること」だけでなく**「絶対やるな」**の列がある。

なぜか。27施設のアンケートとケーススタディを分析すると、**災害時の二次被害の多くは「やりすぎ」から生まれていた**からだ。

| やりがちなこと | なぜ危険か |
|-------------|----------|
| 1人で患者搬送を始める | 転倒・落下で患者も自分も負傷する |
| 素人が電気設備に触る | 感電・短絡でさらに停電範囲が拡大 |
| 受水槽の残量を確認せず楽観する | 残量ゼロになってから慌てても手遅れ |
| ガス配管を自力修理しようとする | ガス漏れを起こし、病棟全体が危険に |
| 電話がつながるまで待ち続ける | 最初の10分が情報ゼロで過ぎる |

**「やらない」判断も、立派な初動対応だ。**

免責条項は「やった結果」だけでなく、「やらなかった判断」も保護します。1人では搬送できないから待った。修理できないから触らなかった。これらは正しい判断だ。

## 第5章：導入ロードマップ

### Step 1: 院長の署名を取る（最優先）

免責条項に院長が署名するだけで、組織の空気が変わります。

```
やること:
1. 免責条項の文面を防災委員会に提出
2. 院長の署名を取得
3. 署名入りカードをコピーし、全配備場所に設置

所要時間: 防災委員会1回分
必要な人: 院長、防災委員長
```

これが全ての出発点です。カードの中身の改善は後でいい。免責の署名が先だ。

### Step 2: 超軽量カードを配備する

- A5サイズ片面で印刷（ラミネート推奨）
- 配備場所：災害BOX内、当直室、ナースステーション
- スマートフォン用PDF を院内共有フォルダ・院内グループウェアに配置
- **院長署名入りのSECTION 0を必ず含める**

### Step 3: 当直者オリエンテーション（3分間）

新任当直者のオリエンテーションに追加する内容：

1. 「このカードがあなたの権限証明書です」と伝える（30秒）
2. SECTION 0（免責条項）を一緒に読む（60秒）
3. 5項目チェックを声に出して読む（60秒）
4. 「迷ったら宣言。空振りOK」を復唱（30秒）

**マニュアル100ページの研修より、この3分のほうが効果があります。**

### Step 4: 夜間抜き打ち訓練

- シナリオ：「午前2時、震度6、停電」
- カードだけを頼りに10分間の初動を実施
- 訓練後の振り返りポイント：
  - 発災宣言をためらわなかったか？
  - 5項目チェックは5分以内に完了したか？
  - 「院長に確認してから」と言った人はいなかったか？
  - 免責条項の存在を知っていたか？

### Step 5: 振り返りと改善

- 訓練後、「カードで分かりにくかった点」をフィードバック
- 年1回、カードの内容を見直し（バージョン管理）
- **ただし、免責条項とSECTION 0は変えない。ここが土台。**

## 第6章：よくある反論と回答

### 「免責条項なんて法的に有効なのか？」

労働法上、免責条項が裁判所で直接的な法的拘束力を持つわけではない。しかし重要なのは、**組織の意思表示としての効力**です。事後検証において「院長が署名した方針に基づいた行動だった」と示せることは、当事者にとって大きな盾になります。また実務上、院長署名の免責条項を持つ病院で、善意の発災宣言に対して懲戒処分が行われた事例は確認されていない。

### 「空振りが多すぎたらどうする？」

空振りの記録はクロノロジーに残ります。空振りが頻発する場合、それは「発災基準が曖昧」という別の問題です。基準を明確にすることで空振り頻度は下がります。しかし、空振りを罰することは絶対にしない。罰した瞬間、誰も宣言しなくなります。

### 「5項目では足りない」

足りません。しかし最初の10分に必要十分なのはこの5つです。10分後にはPhase 1の増援が到着し始めます。詳細点検はそこから。最初の10分を乗り切ることが全ての前提だ。

### 「院長がサインしてくれない」

そうであれば、まず問いかけてください。「夜間に震度6が来て、当直医が発災宣言を出せず、初動が15分遅れた場合、責任は誰にありますか？」と。免責条項に署名しないリスクのほうが、署名するリスクよりはるかに大きいことを示すべきだ。

---

## おわりに：恐怖を設計で解く

災害対応における最大の障壁は、設備でも知識でもない。

**「判断して大丈夫か？」という恐怖**だ。

超軽量アクションカード v2.0は、この恐怖に対する設計上の回答だ。

- **免責条項**が「判断していい」と保証する
- **5項目チェック**が「何を見ればいい」を教える
- **10分タイムライン**が「いつ何をする」を示す
- **「やるな」リスト**が「やりすぎ」を止める

すべては、深夜2時に1人で立つ当直医の背中を押すために。

**あなたの善意の判断を、組織が守る。**

この一文が書かれたカードが、全国の病院の当直室に置かれる日を目指している。
