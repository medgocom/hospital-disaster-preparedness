# 「赤はどこに置く？」が決められない病院は、トリアージ訓練をしても無駄です

**〜27施設のアンケートが暴いた「ゾーニング崩壊の4層構造」と、物理・動線・時間・合意で設計する現実解〜**

---

## はじめに：テープは買った。でも貼る場所が決まっていない

全国27施設の病院防災アンケートで、災害BOXの整備が進んでいます。中には赤・黄・緑のゾーニングテープが入っています。トリアージタグも入っている。

では、質問だ。

**そのテープを「どこに」貼るか、今この瞬間に答えられますか？**

答えられないなら、この記事はあなたの施設のためのものだ。

27施設のトリアージ演習フィードバック（アンケート質問3）を分析した結果、現場が最も苦しんでいたのは「START法を覚えること」ではありませんだった。

**「自分の病院の、この廊下の、どこに線を引くか」**——この一点だ。

そして分析を深めると、ゾーニングが崩壊する原因は、従来の「重力・光・心理」の3原則では捉えきれない**4つの層**で構造化できることが分かりた。

---

## 第1章：ゾーニング崩壊の「4層構造」

27施設の失敗パターンを再構造化した結果、ゾーニングが崩壊する原因は4つの独立した層に分類されます。

```
┌─────────────────────────────────────────────────────┐
│         ゾーニング崩壊の4層構造                       │
│                                                       │
│  第1層: 物理（Physics）                              │
│    建物の構造・重力・搬送経路の制約                    │
│    → エレベーター停止で垂直移動が不可能に              │
│                                                       │
│  第2層: 動線（Flow）                                 │
│    人流の交差・視認性・遮蔽すべき経路                  │
│    → 4つの人流が1箇所で衝突。黒エリアの動線が空白     │
│                                                       │
│  第3層: 時間（Time）                                 │
│    30分後の患者増加・エリア飽和・長期化                │
│    → 最初に引いた線は30分後に使えない                  │
│                                                       │
│  第4層: 合意（Consensus）                            │
│    職種間の価値対立・役割の曖昧さ・心理的障壁          │
│    → 医師・看護師・事務が「赤の場所」で揉める          │
│                                                       │
└─────────────────────────────────────────────────────┘
```

従来の「重力・光・心理」の3原則は、この4層に再編されます。「重力」は第1層（物理）に、「光」は第2層（動線）に、「心理」は第4層（合意）に対応し、見落とされていた第3層（時間）が独立した層として浮上した。

以下、各層の崩壊パターンを、札幌東徳洲会病院・茅ヶ崎徳洲会病院・湘南鎌倉総合病院の具体的知見とともに解説する。

## 第2章：第1層——物理の壁（重力に逆らう配置は崩壊する）

### 崩壊パターン：「赤エリアは手術室の近くに」の罠

**現場の声：**
> 「Ⅰ・Ⅱ・Ⅲのエリア区分やどこでトリアージすべきか」
> 「トリアージエリアの設置場所」

多くの施設が「赤エリアは手術室の近くに」と考えます。論理的には正しい。しかし手術室が3階にある場合、**エレベーターが停止した瞬間にその配置は崩壊します。**

ストレッチャーに乗った重症患者を、夜勤看護師3名で3階まで階段搬送する。1人の搬送に4名・15分。物理的に成立しない。

### 札幌東徳洲会病院の教訓：「動かない設備」の連鎖

2018年北海道胆振東部地震（ブラックアウト）で、札幌東徳洲会病院は以下の事態に直面した。

- コージェネレーションシステムが稼働しなかった
- 非常用コンセント（赤コンセント）が使えなかった
- 全道295万戸の停電により、**エレベーターも暖房も同時に停止**

寒冷地では停電＝暖房停止＝生命危機です。エレベーター停止と暖房停止が同時に起きると、**患者を垂直に動かせないうえに、低体温で状態が悪化する**という二重の物理的制約が発生する。

### 設計原則：下に置け、上に運ぶな

```
【物理層の原則】
トリアージポスト → 1F（地上階）
赤エリア         → 処置室と同一フロア、または搬入口と同一フロア
緑エリア         → 自力歩行で到達可能な場所
黒エリア         → 1Fの目立たない場所（搬送の負担を最小化）

エレベーターは「動いたらラッキー」程度の想定とする。
手術室が上層階の場合 → 「手術機能を1Fに仮設する」ほうが現実的。
```

27施設中、「エレベーター停止時のゾーニング代替案」を持っている施設はほとんどありませんでした。図面上で引いた動線は、停電の瞬間に重力に敗北する。

## 第3章：第2層——動線の壁（見える動線と見せてはいけない動線）

### 崩壊パターン：4つの人流が1箇所で衝突する

**現場の声：**
> 「出入口の明確化、動線」
> 「トリアージの動線の明確化」

停電時、院内は非常灯のぼんやりした光だけになります。その暗闘の中で、搬入されてくる傷病者、避難する入院患者、駆けつけた家族、そして帰宅困難者が**同じ玄関に殺到**する。

```
→→→ 動線A: 搬入動線（外から傷病者が入る）
←←← 動線B: 避難動線（院内患者が出る）
→→→ 動線C: 家族・帰宅困難者（探しに来る）
↔↔↔ 動線D: 医療者（エリア間を移動する）
```

一方通行にしたくても、物理的な封鎖手段（バリケード、ロープ、テープ）の設置場所が事前に決まっていない。テープは災害BOXにある。でも**「どの柱とどの柱の間に貼るか」が決まっていない**。

### 黒エリアの動線設計：最もセンシティブな空白地帯

演習で最も議論が避けられるのが**黒エリア（遺体安置場所）の動線**だ。

NotebookLMの分析から：
> 「黒エリアへの搬送ルートを確保しつつ、他の患者や家族の目に触れさせない動線を確保することは、既存の施設構造上極めて困難」

黒エリアの配置は、医学的な合理性だけでなく**倫理的・心理的な配慮**が要る。しかしあまりにセンシティブなテーマであるため、演習で「では黒エリアはどこにしますか？」と言い出せる人がいない。結果、最もクリティカルなエリアの動線が空白のまま残ります。

これは27施設に共通する空白地帯だ。

### 設計原則：4つの動線を物理的に分離する

```
【動線層の原則】
① 搬入動線（動線A）: 救急入口を使用
② 避難動線（動線B）: 正面玄関 or 非常口を使用
③ 家族動線（動線C）: 別の入口 or 待機エリアへ誘導
④ 黒エリア動線（動線D）: 裏口・地下通路など非公開経路

動線AとBが同じ出入口を使う場合 → 時間差または物理的封鎖で分離
黒エリアの搬送ルート → 図面上でのみ共有（掲示しない）
```

視認性の事前整備として：
- 蓄光テープを各エリアの入口に貼っておく（停電対応）
- 搬入口と避難口を事前に決定し、図面に明記
- 黒エリアへの搬送ルートは、建物裏口や地下通路など「見えにくい経路」を選定

## 第4章：第3層——時間の壁（30分後、線が使えなくなる）

### 崩壊パターン：初期配置だけで「計画済み」と思い込む

**現場の声：**
> 「2回目以降はどのような流れになるのか」
> 「トリアージは一度したら次回確認するのはどれくらい時間をあけた方がいいのか」

初回のゾーニング（一次トリアージの配置）は研修で学べます。問題は**30分後**だ。

| 時間 | 現実に起きること | ゾーニングへの影響 |
|------|----------------|-----------------|
| 0〜30分 | 傷病者がトリアージポストに到着し始める | 初期配置で対応可能 |
| 30分〜2時間 | 患者数が急増。赤エリアが溢れる | **赤エリアの拡張が必要。隣のエリアを侵食する** |
| 2〜6時間 | 増援到着。二次トリアージ開始 | **赤の一部を「手術待ち」「ICU待ち」に転換** |
| 6〜12時間 | 入院患者との統合が必要に | **通常病棟のベッドとトリアージエリアの境界が消滅** |
| 12時間〜 | 長期化。スタッフの交代 | **ゾーニング自体の解除基準が不明** |

27施設のアンケートでは、**再トリアージのタイミング基準が標準化されている施設はゼロ**でした。「15分ごとに再評価」なのか「状態変化時のみ」なのか、施設内でも統一されていない。

### 設計原則：ゾーニングは「動的」に計画する

```
【時間層の原則】
① 再トリアージのタイミングを明文化する（例：30分ごと or 状態変化時）
② 赤エリアの拡張先を事前に指定する（「溢れたらどこに広げるか」）
③ 二次トリアージ後の動線を定める（赤→手術室、黄→検査、緑→帰宅）
④ 12時間超の長期化を想定したゾーン再編計画を持つ
⑤ ゾーニングの解除基準（通常体制への復帰条件）を定める
```

## 第5章：第4層——合意の壁（医師・看護師・事務、なぜ「赤の場所」で揉めるのか）

### 崩壊パターン：3つの正義が衝突する

| 職種 | 赤エリアの理想位置 | 最重視する価値 |
|------|------------------|-------------|
| 医師 | 処置室・手術室に隣接 | **診療のしやすさ**。モニター・除細動器・挿管セットに手が届く |
| 看護師 | 搬入口に近い場所 | **搬送のしやすさ**。患者の移動回数と距離を最小化したい |
| 事務 | 受付に近い場所 | **記録の起点**。トリアージタグの番号管理、家族対応の窓口 |

3者とも正しい。しかし物理的に全てを満たす場所は存在しない。

### 職種間ギャップの実態：3病院からの知見

#### 札幌東徳洲会病院：「誰がトリアージするのか」

札幌東徳洲会病院のアンケートから、最も素朴で最も深刻な問いが挙がりた。

> 「誰がトリアージするのか」

マニュアル上は「医師」や「認定看護師」です。しかし夜間発災時、医師が到着するまでの空白期間に、トリアージポストは**無人**です。事務職員や若手看護師が「自分がやるべきなのか？」と戸惑っている間に、患者が殺到する。

#### 茅ヶ崎徳洲会病院：「PTでもトリアージできるのか」

> 「PT（理学療法士）でもトリアージの現場でトリアージできるのか」

コメディカルの参加基準が曖昧なまま残されている典型例です。フレームワーク（軸9）では「研修修了者は一次トリアージのみ可」としていますが、この基準自体が現場に周知されていない。

#### 湘南鎌倉総合病院：「計画はある、でも周知されていない」

湘南鎌倉総合病院はトリアージエリアの配置計画を詳細に策定している。

```
マニュアル上の配置：
  トリアージポスト → 医師 or 認定看護師
  誘導係          → 事務員
  記録係          → 事務員
  搬送係          → リハビリスタッフ

現実の発災直後：
  医師 → まだ到着していない
  認定看護師 → 夜勤にいない
  結果 → 事務職員と若手看護師が「やるべきなのか？」と戸惑い、
         ポストが無人のまま患者が殺到する
```

**仕組みは存在するが、文化として定着していない。** 前回の記事（[なぜ病院のマニュアルは「ゴミ」になるのか？](https://github.com/medgocom/hospital-disaster-preparedness/blob/main/docs/articles/note-003-real-bcp.md)）で指摘した「空文化」の典型だ。

### 設計原則：合意を「プロセス」で作る

#### 3職種合同の「図面ワーク」

赤エリアの配置を「正解」から始めるのではなく、**「3職種が同時に図面を囲む」ことから始めます。**

```
【図面ワークの手順】（所要60分）

Step 1: 白紙の図面を3部コピーする（15分）
  → 医師、看護師代表、事務代表にそれぞれ配布
  → 各自が「理想のゾーニング」を赤ペンで書き込む

Step 2: 3枚を重ねて比較する（15分）
  → 一致した場所 = 全員が合意できるエリア
  → 不一致の場所 = 議論すべきポイント

Step 3: 「物理層の原則」で制約を加える（15分）
  → エレベーター停止を前提に、上層階のエリアを再検討
  → 残った選択肢の中から、3者の妥協点を探す

Step 4: 決定事項を写真付きで記録する（15分）
  → 「この柱とこの柱の間が赤エリア」レベルの具体性で
  → 写真を撮り、アクションカードに添付
```

**重要: このプロセス自体が合意形成です。** 結果の配置図よりも、3職種が「一緒に決めた」という事実のほうが価値がある。

#### 「START法で緑。でも明らかに重症。」をどうするか

> 「START法で緑でも明らかに重症者である場合などどうするか」
> 「自分の感情をコントロールできない人（小さい子供や知的障害がある人）のトリアージ」

| ケース | START法の判定 | 臨床的な直感 | 対処 |
|-------|-------------|------------|------|
| 歩行可能だが顔面蒼白・冷汗 | 緑（歩行可能） | 赤（内出血疑い） | **要再評価フラグ** |
| 泣き叫ぶ小児（外傷なし） | 緑（歩行可能・呼吸正常） | 黄（精神的ケア必要） | **心理フラグ** |
| 知的障害者（指示に従えない） | 判定不能（協力が得られない） | — | **個別対応フラグ** |

#### フラグ制度の導入

```
【フラグ制度】
  START法の判定: 通常通り赤・黄・緑・黒で判定（速度を維持）

  フラグ: START法の判定に加え、以下の場合にフラグを付ける
  ・臨床的な直感と判定が矛盾する → 「要再評価フラグ」
  ・判定に協力が得られない → 「個別対応フラグ」
  ・心理的ケアが必要 → 「心理フラグ」

  フラグ付き患者は、15分以内に医師が再評価する
```

トリアージポストの看護師は「私の判断で赤に上げていいのか」と迷わなくてよい。フラグを付けて緑エリアに送り、15分以内に医師が来る。これで判断の凍結を防げます。

## 第6章：実力診断シートで自施設を測る

ここまでの「4層構造」の分析を、自施設で実践するための**「トリアージ・ゾーニング実力診断シート」**を設計した。

### 構成

| PART | 診断対象 | 項目数 |
|------|---------|-------|
| 物理（Physical） | 建物構造・搬送経路・フロア配置 | 7 |
| 動線（Flow） | 視認性・動線交差・遮蔽 | 7 |
| 時間（Time） | 再トリアージ・拡張計画・長期化 | 6 |
| 合意（Consensus） | 役割定義・職種間合意・判断基準 | 7 |
| **合計** | | **27** |

27項目。**27施設のアンケートにちなんだ数**だ。

各項目に◯/△/×をつけるだけで、自施設のゾーニング体制の「実力」が可視化されます。

### 使い方のポイント

1. **防災委員会の席上で、図面を広げながら実施する**（デスクワークではなく）
2. **「×」が付いた項目を3つ選び、次回訓練の重点項目にする**（全部直そうとしない）
3. **年1回、同じシートで再診断する**（改善の可視化）

テンプレートはGitHubで公開している。

## おわりに：図面の上の線は、足で引け

27施設のアンケートが教えてくれたのは、「トリアージの知識」と「自施設でのゾーニング」の間にある深い溝だった。

START法は教室で学べます。赤・黄・緑の概念は座学で理解でくる。

しかし、**「この廊下の、この柱と、あの柱の間に赤テープを貼る」**——この一歩は、自分の足で施設を歩かなければ踏み出せない。

ゾーニングが崩壊する原因は、1つではない。4つの層が重なっている。

- **物理の壁**が教えてくれるのは、「上に運べない」という制約
- **動線の壁**が教えてくれるのは、「見える動線」と「見せてはいけない動線」
- **時間の壁**が教えてくれるのは、「最初の線は30分しか持たない」という現実
- **合意の壁**が教えてくれるのは、「人は迷い、怖がり、そして対立する」という事実

この4つの壁と向き合い、自施設の図面の上に線を引く。その線は、座学の綺麗な図面ではなく、**廊下を歩いた足の記憶から生まれる線**だ。

次の防災委員会で、ぜひ図面を広げてください。医師と看護師と事務職員が一緒に。そして1Fのロビーに立って、赤テープを貼る場所を、3人で決めろ。

**「赤はどこに置く？」——その問いに答えられる病院は、もう強い。**
