# 軸3: 医療ガス（供給停止対応）

## 概要

医療ガス（特に酸素）の供給停止は、呼吸器疾患患者や手術中の患者の生命に直結する緊急事態です。配管系統の理解と携帯ボンベへの切替手順の習熟が重要です。

## リスク評価

### 医療ガスの種類と影響

| ガス種別 | 主な使用場所 | 途絶時の影響 | 代替手段 |
|---------|------------|-------------|---------|
| 酸素（O2） | 全病棟・手術室・ICU | 致命的 | 携帯ボンベ |
| 笑気（N2O） | 手術室 | 麻酔方法変更 | 静脈麻酔 |
| 圧縮空気 | 人工呼吸器・手術機器 | 重大 | コンプレッサー |
| 吸引（バキューム） | 全病棟・手術室 | 重大 | 手動吸引器 |
| 窒素（N2） | 手術機器駆動 | 中 | 電動器具切替 |

### 自施設の医療ガス体制

| 項目 | 内容 |

## 対応フロー

### Phase 1: 初動（0〜5分）

```
医療ガス供給異常
    ↓
アラーム確認・原因特定
    ↓
[配管系統問題]        [供給元問題]
    ↓                    ↓
遮断弁操作          業者連絡
バイパス切替        ボンベ切替準備
```

### Phase 2: ボンベ運用（5分〜）

1. **酸素使用患者の把握**
   - 人工呼吸器患者数
   - 酸素療法患者数（流量別）
   - 手術室使用状況

2. **ボンベ配布優先順位**
   - ICU/CCU（最優先）
   - 手術室（進行中手術）
   - 高流量酸素使用患者
   - 低流量酸素使用患者

3. **酸素消費量の計算と残量管理**

### Phase 3: 継続対応

- 酸素使用量の制限検討
- 不急の手術延期
- 転院の検討
- 追加ボンベの調達

## 酸素ボンベ使用時間早見表

### 500Lボンベ（満充填時）

| 流量 | 使用可能時間 |
|-----|-------------|
| 1L/分 | 約8時間 |
| 2L/分 | 約4時間 |
| 3L/分 | 約2時間40分 |
| 5L/分 | 約1時間40分 |
| 10L/分 | 約50分 |

### 計算式

使用時間（分） = ボンベ容量（L） × (現在圧力/充填圧力) ÷ 流量（L/分）

## 実例と教訓（ケーススタディ）

### Case 1: 電源喪失時の医療ガス供給への影響

**関連事例**: 熊本赤十字病院（2016年熊本地震）

**リスク**:
- 電動式の医療ガス供給システム（コンプレッサー、真空ポンプ）が停止
- 人工呼吸器患者への酸素供給に影響
- 麻酔器、吸引器などが使用不能

**教訓**:
1. 医療ガス供給設備の電源を**非常用電源に接続**
2. ポータブル酸素ボンベの十分な備蓄
3. 手動式吸引器の配備
4. 電源復旧の優先順位で医療ガス関連設備を上位に設定

**チェックポイント**:
- [ ] 医療ガス供給設備は非常用電源に接続されているか
- [ ] ポータブル酸素ボンベは各病棟に配備されているか
- [ ] 手動式吸引器の操作訓練を実施しているか
