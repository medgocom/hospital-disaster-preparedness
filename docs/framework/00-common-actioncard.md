# 全軸共通: アクションカード・災害BOX標準セット

## 概要

アンケート結果から、**全27施設で最も関心が高かった**のが「アクションカード・マニュアルの整備」と「災害BOXの設置・活用」でした。本ドキュメントは8軸全てに共通する基盤として、標準的なアクションカードの構成と災害BOXの内容を定義します。

## アクションカードとは

災害発生時に「誰が」「何を」「どの順番で」行うかを、**1枚のカードで即座に判断できる**ようにしたツールです。

### アクションカードの3原則

1. **30秒で理解できる** - 文字は最小限、フロー図中心
2. **判断を迷わせない** - Yes/Noで分岐、曖昧な表現を排除
3. **その場で完結する** - 必要な連絡先・物品場所を記載

## 発災スイッチ（災害宣言基準）

### アンケートで挙がった課題

> 「災害のスイッチ（発災宣言）の入れ方・基準の明確化」
> 「夜間・休日（幹部不在時）の組織図作成」
> 「誰でもスイッチを入れられるようにする」

### 発災スイッチの基準例

| レベル | 基準 | 宣言者 | アクション |
|-------|-----|-------|----------|
| **レベル1**（注意） | 震度4以上 / 大雨警報 | 当直責任者 | 情報収集開始 |
| **レベル2**（警戒） | 震度5弱以上 / インフラ1つ途絶 | 当直責任者 | 災害対策本部準備 |
| **レベル3**（災害） | 震度5強以上 / インフラ2つ以上途絶 / 多数傷病者 | 誰でも可 | 災害対策本部設置 |

### 夜間・休日の組織体制

```
【平日日勤】          【夜間・休日】
  院長                  当直医師
    │                      │
  副院長              当直看護師長
    │                      │
各部門長              各部署リーダー
```

**重要**: 夜間・休日は「当直医師」が最高意思決定者となる。
判断に迷う場合は**レベルを上げる方向**で宣言する。
