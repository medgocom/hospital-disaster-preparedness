# 軸6: エレベーター（停止時対応）

## 概要

エレベーター停止は、特に高層病院において患者搬送に重大な支障をきたします。階段搬送の体制整備と、閉じ込め時の救出手順を準備しておく必要があります。

## リスク評価

### エレベーター停止の影響

| 影響 | 深刻度 | 対応策 |
|-----|-------|-------|
| 救急患者搬送不可 | 重大 | 階段搬送体制 |
| 手術室への搬送不可 | 重大 | 手術スケジュール調整 |
| 透析患者の移動困難 | 重大 | 透析室フロア調整 |
| 給食配膳不可 | 中 | 各階調理・配膳 |
| 物品搬送不可 | 中 | 人力リレー搬送 |
| 閉じ込め発生 | 緊急 | 救出手順 |

### 自施設のエレベーター体制

| 項目 | 内容 |
|-----|------|
| エレベーター台数 | 台（うち患者用　台） |
| 非常用電源対応 | 台 |
| 地震時管制運転 | あり / なし |
| 閉じ込め救出訓練 | 年　回実施 |
| 保守会社 | 会社名：　　緊急連絡先：　　　 |

## 対応フロー

### Phase 1: 閉じ込め対応（即時）

```
エレベーター停止
    ↓
閉じ込め有無の確認
    ↓
[閉じ込めあり]
    ↓
インターホンで状況確認
    ↓
保守会社へ連絡
    ↓
[救出に時間がかかる場合]
    ↓
院内救出チーム対応
```

### Phase 2: 搬送体制構築（15分〜）

1. **階段搬送チームの編成**
   - 各フロア2〜4名配置
   - リーダー任命
   - 搬送器具の配備

2. **搬送優先順位の決定**
   - 緊急手術患者
   - 重症患者の検査移動
   - 透析患者

3. **物品搬送リレーの構築**

### Phase 3: 継続対応

- 不要な患者移動の抑制
- 手術スケジュールの見直し
- 外来診療の調整

## 階段搬送の方法

### 必要機材

| 機材 | 用途 | 配置場所 |
|-----|------|---------|
| ストレッチャー担架 | 臥床患者搬送 | 各階段踊り場 |
| 布担架 | 軽量患者搬送 | 各病棟 |
| 車椅子 | 座位可能患者 | 各病棟 |
| 階段避難車 | 車椅子患者 | 各フロア |

### 搬送手順

1. **下り搬送（基本）**
   - 頭側を上に
   - 4名体制（前2名、後2名）
   - 声掛け統一「いち、に、いち、に」
   - 踊り場で休憩・交代

2. **上り搬送**
   - 頭側を上に維持
   - 6名体制推奨
   - 短い距離で交代

### 安全上の注意

- 患者の状態を常に確認
- 点滴・ドレーン類の管理
- 転落防止の固定確認
- 搬送者の体力管理
