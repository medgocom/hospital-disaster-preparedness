# 軸8: 給食・廃棄物・燃料

## 概要

この軸では、病院運営を支える3つの重要なロジスティクスを扱います。
1. **給食**: 患者・職員の食事確保
2. **廃棄物**: 医療廃棄物・一般廃棄物の適正処理
3. **燃料**: 非常用発電機・ボイラー等の燃料確保

## 1. 給食（食事確保）

### リスク評価

| 状況 | 影響 | 対応策 |
|-----|-----|-------|
| 停電 | 調理不可・冷蔵庫停止 | 備蓄食・外部調達 |
| 断水 | 調理・洗浄不可 | 備蓄食・使い捨て食器 |
| ガス停止 | 加熱調理不可 | 非加熱食・カセットコンロ |
| 物流途絶 | 食材入手不可 | 備蓄食 |
| 給食委託先被災 | 給食提供不可 | 自前調理・外部調達 |

### 備蓄食の考え方

| 区分 | 対象 | 備蓄量目安 | 備考 |
|-----|-----|----------|------|
| 入院患者 | 全入院患者 | 3日分以上 | 特殊食対応も含む |
| 職員 | 災害時出勤者 | 3日分 | |
| 帰宅困難者 | 想定人数 | 1日分 | |

### 特殊食への対応

- 経管栄養（濃厚流動食）の備蓄
- アレルギー対応食
- 嚥下調整食（ソフト食等）
- 糖尿病食・腎臓病食

### 対応フロー

```
給食提供困難
    ↓
備蓄食での対応開始
    ↓
外部調達の検討
    ↓
[長期化の場合]
    ↓
自治体・他施設への支援要請
```

## 2. 廃棄物（適正処理）

### リスク評価

| 状況 | 影響 | 対応策 |
|-----|-----|-------|
| 収集業者被災 | 廃棄物滞留 | 院内保管場所確保 |
| 感染性廃棄物増加 | 処理能力超過 | 保管期間延長 |
| 処理施設被災 | 処理先なし | 広域連携 |

### 災害時の廃棄物管理

1. **一時保管場所の確保**
   - 屋外保管場所の設定
   - 感染性廃棄物は冷暗所
   - 動線との分離

2. **分別の徹底**
   - 感染性廃棄物
   - 一般廃棄物
   - 資源ごみ

3. **保管期間の目安**
   - 感染性廃棄物: 冷蔵なし72時間以内
   - 一般廃棄物: 1週間程度

### 感染対策

- 廃棄物取扱者のPPE着用
- 廃棄物容器の二重化
- 保管場所の消毒

## 3. 燃料（エネルギー確保）

### 燃料を使用する設備

| 設備 | 燃料種類 | 消費量目安 | 備蓄量 |
|-----|---------|----------|-------|
| 非常用発電機 | 重油/軽油 | L/時間 | L（　時間分） |
| ボイラー | 重油/ガス | L/時間 | L |
| 厨房 | プロパン/都市ガス | kg/日 | kg |
| 公用車 | ガソリン/軽油 | - | - |

### 燃料調達ルート

| 優先順位 | 調達先 | 連絡先 | 備考 |
|---------|-------|-------|------|
| 1 | 契約業者 | | 24時間対応可否 |
| 2 | 近隣ガソリンスタンド | | 災害時協定 |
| 3 | 自治体 | | 緊急時要請ルート |
| 4 | 他病院からの融通 | | グループ内連携 |

### 対応フロー

```
燃料残量低下（残り24時間分）
    ↓
契約業者へ緊急配送依頼
    ↓
[配送不可の場合]
    ↓
代替調達先への連絡
    ↓
[調達不可の場合]
    ↓
発電機使用制限・優先給電
```

## 地域別の追加対策

### 離島：長期孤立を見据えた備蓄

離島では物資輸送が途絶した場合、外部からの支援到着まで数日〜1週間以上かかる可能性があります。

#### アンケートで挙がった声

> 「非常食」や「災害BOX」への関心が高い
> 「栄養科だけでなく全スタッフへの周知」
> 「歯科用備品（歯ブラシ等）の確保」

#### 離島向け備蓄量の目安

| 区分 | 通常目安 | 離島推奨 | 備考 |
|-----|---------|---------|------|
| 食料 | 3日分 | **7日分以上** | 職員家族分も考慮 |
| 飲料水 | 3日分 | **7日分以上** | 1人3L/日 |
| 医薬品 | 3日分 | **14日分** | 特に透析・インスリン |
| 燃料 | 3日分 | **7日分以上** | 発電機・ボイラー用 |

#### 離島特有の備蓄品目

- [ ] 歯ブラシ・歯磨き粉（長期化対応）
- [ ] ウェットティッシュ（大量）
- [ ] 生理用品
- [ ] おむつ（乳児用・成人用）
- [ ] 粉ミルク・離乳食
- [ ] 常備薬（鎮痛剤・胃薬等）

#### 職員家族分の備蓄

離島では職員が病院に泊まり込む場合、家族も病院に避難してくる可能性があります。

| 対象 | 人数想定 | 備蓄内容 |
|-----|---------|---------|
| 職員 | 全職員の50% | 食料・飲料水・寝具 |
| 職員家族 | 職員の1.5倍 | 食料・飲料水 |
| 近隣住民 | 施設による | 最低限の食料・水 |

### 寒冷地：冬季の燃料確保

#### 寒冷地特有のリスク

- 暖房用燃料の消費量が急増
- 積雪による燃料配送の遅延
- 凍結による配管トラブル

#### 冬季の燃料備蓄目安

| 燃料 | 通常目安 | 冬季推奨 |
|-----|---------|---------|
| 暖房用重油 | 3日分 | **7日分以上** |
| 発電機用軽油 | 3日分 | **5日分以上** |
| プロパンガス | 3日分 | **5日分以上** |

### 都市部：帰宅困難者対応

大規模災害時、都市部の病院には帰宅困難者が押し寄せる可能性があります。

#### 想定人数の算出

```
帰宅困難者想定 = 周辺就業者数 × 帰宅困難率（約30%） × 収容割合（約5%）
```

#### 帰宅困難者向け備蓄

- [ ] 飲料水（1人500ml × 想定人数）
- [ ] 乾パン・クラッカー
- [ ] アルミブランケット
- [ ] 簡易トイレ
