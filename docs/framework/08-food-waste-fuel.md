# 軸8: 給食・廃棄物・燃料

## 概要

この軸では、病院運営を支える3つの重要なロジスティクスを扱います。
1. **給食**: 患者・職員の食事確保
2. **廃棄物**: 医療廃棄物・一般廃棄物の適正処理
3. **燃料**: 非常用発電機・ボイラー等の燃料確保

## 1. 給食（食事確保）

### リスク評価

| 状況 | 影響 | 対応策 |
|-----|-----|-------|
| 停電 | 調理不可・冷蔵庫停止 | 備蓄食・外部調達 |
| 断水 | 調理・洗浄不可 | 備蓄食・使い捨て食器 |
| ガス停止 | 加熱調理不可 | 非加熱食・カセットコンロ |
| 物流途絶 | 食材入手不可 | 備蓄食 |
| 給食委託先被災 | 給食提供不可 | 自前調理・外部調達 |

### 備蓄食の考え方

| 区分 | 対象 | 備蓄量目安 | 備考 |
|-----|-----|----------|------|
| 入院患者 | 全入院患者 | 3日分以上 | 特殊食対応も含む |
| 職員 | 災害時出勤者 | 3日分 | |
| 帰宅困難者 | 想定人数 | 1日分 | |

### 特殊食への対応

- 経管栄養（濃厚流動食）の備蓄
- アレルギー対応食
- 嚥下調整食（ソフト食等）
- 糖尿病食・腎臓病食

### 対応フロー

```
給食提供困難
    ↓
備蓄食での対応開始
    ↓
外部調達の検討
    ↓
[長期化の場合]
    ↓
自治体・他施設への支援要請
```

## 2. 廃棄物（適正処理）

### リスク評価

| 状況 | 影響 | 対応策 |
|-----|-----|-------|
| 収集業者被災 | 廃棄物滞留 | 院内保管場所確保 |
| 感染性廃棄物増加 | 処理能力超過 | 保管期間延長 |
| 処理施設被災 | 処理先なし | 広域連携 |

### 災害時の廃棄物管理

1. **一時保管場所の確保**
   - 屋外保管場所の設定
   - 感染性廃棄物は冷暗所
   - 動線との分離

2. **分別の徹底**
   - 感染性廃棄物
   - 一般廃棄物
   - 資源ごみ

3. **保管期間の目安**
   - 感染性廃棄物: 冷蔵なし72時間以内
   - 一般廃棄物: 1週間程度

### 感染対策

- 廃棄物取扱者のPPE着用
- 廃棄物容器の二重化
- 保管場所の消毒

## 3. 燃料（エネルギー確保）

### 燃料を使用する設備

| 設備 | 燃料種類 | 消費量目安 | 備蓄量 |
|-----|---------|----------|-------|
| 非常用発電機 | 重油/軽油 | L/時間 | L（　時間分） |
| ボイラー | 重油/ガス | L/時間 | L |
| 厨房 | プロパン/都市ガス | kg/日 | kg |
| 公用車 | ガソリン/軽油 | - | - |

### 燃料調達ルート

| 優先順位 | 調達先 | 連絡先 | 備考 |
|---------|-------|-------|------|
| 1 | 契約業者 | | 24時間対応可否 |
| 2 | 近隣ガソリンスタンド | | 災害時協定 |
| 3 | 自治体 | | 緊急時要請ルート |
| 4 | 他病院からの融通 | | グループ内連携 |

### 対応フロー

```
燃料残量低下（残り24時間分）
    ↓
契約業者へ緊急配送依頼
    ↓
[配送不可の場合]
    ↓
代替調達先への連絡
    ↓
[調達不可の場合]
    ↓
発電機使用制限・優先給電
```
