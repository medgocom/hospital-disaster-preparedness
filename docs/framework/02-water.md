# 軸2: 水道（断水対応）

## 概要

断水は医療行為の継続に重大な影響を与えます。手洗い、滅菌、透析、給食など、水は病院運営のあらゆる場面で必要です。

## リスク評価

### 水の使用用途と優先度

| 優先度 | 用途 | 1日使用量目安 | 代替手段 |
|-------|-----|-------------|---------|
| 最優先 | 飲料水 | 3L/人/日 | ペットボトル備蓄 |
| 最優先 | 透析用水 | 120L/回/人 | 給水車、井戸水RO処理 |
| 優先 | 手指消毒 | - | アルコール消毒 |
| 優先 | 医療器材洗浄 | - | ディスポ製品切替 |
| 通常 | トイレ洗浄 | - | 簡易トイレ |
| 通常 | 入浴 | - | 清拭対応 |
| 通常 | 給食調理 | - | 備蓄食・外部調達 |

### 自施設の給水体制

| 項目 | 内容 |
|-----|------|
| 受水槽容量 | m³（約　日分） |
| 高架水槽容量 | m³ |
| 井戸設備 | あり / なし |
| 給水車受入設備 | あり / なし |
| 飲料水備蓄 | L（　日分） |

## 対応フロー

### Phase 1: 初動（0〜30分）

```
断水発生
    ↓
受水槽残量確認
    ↓
節水指示発令
    ↓
優先使用エリア決定
```

### Phase 2: 節水対応（30分〜）

1. **即時節水措置**
   - トイレ洗浄の制限
   - シャワー・入浴の中止
   - 不要な水使用の禁止

2. **代替手段への切替**
   - 手洗い → アルコール消毒
   - 食器 → 使い捨て容器
   - 清拭 → ドライシャンプー等

3. **透析患者対応の決定**
   - 透析継続可否の判断
   - 転院手配（必要時）

### Phase 3: 外部調達（数時間〜）

- 給水車の要請（行政・自衛隊）
- ペットボトル水の追加調達
- 井戸水の活用検討

## 実例と教訓（ケーススタディ）

### Case 1: 井水設備を持たない施設群の共通課題

**対象施設例**: 国立病院機構千葉医療センター、帝京大学医学部付属病院、東北労災病院、仙台オープン病院、久留米大学医療センター、済生会横浜市東部病院 など

**リスク**:
- 上水道が途絶した場合、**代替水源がない**
- 透析、手術、滅菌など水を大量に使用する医療行為が制限される
- 飲料水・生活用水の確保も困難になる

**推奨対策**:

| 期間 | 対策 |
|-----|------|
| **短期** | 給水車の手配先リスト作成（複数業者）<br>ペットボトル水の備蓄（最低3日分、可能なら7日分）<br>近隣の井戸保有施設との協定 |
| **中期** | 貯水槽の増設または大型化<br>雨水貯留システムの導入（トイレ洗浄用など） |
| **長期** | 井水設備の新設検討<br>地下水脈調査の実施 |

**タグ**: `#井水なし` `#断水脆弱` `#代替水源なし`

---

### Case 2: 東北労災病院（貯水制限）

**状況**: 上水貯留が通常時の50%×3日分と制限付き

**リスク**:
- 災害時に使用量が増加した場合、3日持たない可能性
- 復旧が長期化した場合の対応が困難

**教訓**:
1. 貯水量は「**通常時の100%×3日分**」を目標とする
2. 災害時の節水プロトコルを事前に策定
3. 優先度の高い用途（透析、手術、飲料）と低い用途（清掃、洗濯）を明確化

**タグ**: `#貯水制限`

## 夜間・休日対応（少人数での初動）

夜間の断水は発見が遅れやすい。トイレ使用時や手洗い時に「水が出ない」ことで初めて気づくケースが多い。

### 断水発見時の初動フロー（当直者向け）

```
水が出ないことを確認
  ↓
❶ 複数箇所で確認（1F・上階の2箇所以上）
  ↓
❷ 受水槽の水位確認（受水槽室の水位計を目視）
  ↓
  ├─ [水位あり] → ポンプ故障の可能性 → 施設課担当者に電話連絡
  │
  └─ [水位なし] → 上水道の断水 → 節水指示を発令
                   → 水道局に確認（連絡先は災害BOXに記載）
```

### 夜勤者の役割分担

| 役割 | 担当 | やること |
|------|------|---------|
| 指揮・判断 | 当直医師 | 透析患者の対応判断、節水レベルの決定 |
| 患者対応 | 夜勤看護師 | 透析中患者の安全確認、飲料水の配布 |
| 設備確認 | 施設管理当直（不在時は看護師） | 受水槽の水位確認、ポンプ状態の確認 |

### 夜間断水時の3大チェック

1. **透析中の患者はいるか**（いる場合は即座に当直医に報告、透析中止の判断）
2. **受水槽に水は残っているか**（水位計の確認。残量があればポンプ故障を疑う）
3. **トイレ用の水は確保できるか**（バケツ・ポリタンクの場所を確認）

### 夜間の節水プロトコル（即時発動）

| 制限レベル | 内容 | 発動基準 |
|-----------|------|---------|
| レベル1 | シャワー・入浴中止、洗濯中止 | 断水発生時 |
| レベル2 | トイレは簡易トイレに切替、手洗いはアルコール | 受水槽残量50%以下 |
| レベル3 | 飲料水・医療用水のみ使用 | 受水槽残量20%以下 |

> **事前準備**: 受水槽の場所と水位計の見方を当直者に説明しておくこと。簡易トイレの保管場所を全員が把握していること。

## 排水BCP（排水系統の事業継続計画）

従来の断水対策は「給水BCP」——水をどう確保するか——に集中していた。しかし、**「水を出せない」ことも断水と同等のリスク**である。ここでは「排水BCP」として、排水系統の事業継続計画を提案する。

### 排水ポンプの停電リスク

地下に排水ポンプを持つ施設では、停電時にポンプが停止し、下水が逆流するリスクがある。逆流した汚水が地下の電気室・サーバー室を浸水させれば、電力復旧後もシステム復旧ができない二次災害に発展する。

### 排水BCPの4つの柱

| 柱 | 内容 | 確認事項 |
|----|------|---------|
| 排水ポンプ電源 | 排水ポンプの非常電源接続確認 | 電気系統図で確認 |
| 逆流防止 | 逆流防止弁の設置・定期点検 | 実地確認・保守記録 |
| 浸水対策 | 地下設備室の防水（止水板・排水溝） | 実地確認 |
| 使用停止判断 | トイレ使用停止→簡易トイレ切替の基準 | 排水槽水位による判断基準の策定 |

### チェックポイント

- [ ] 排水ポンプの設置場所と台数を把握しているか
- [ ] 排水ポンプは非常用電源に接続されているか
- [ ] 逆流防止弁の設置状況を確認しているか
- [ ] 排水不能時のトイレ使用停止基準があるか

## 揚水ポンプの非常電源確認

### 受水槽に水があっても上階は断水する

受水槽（地下または1階）から上階に水を送る揚水ポンプが非常用電源に接続されていなければ、停電と同時に上階への給水は停止する。高架水槽がある場合も、容量は限定的であり、数時間〜半日で枯渇する。

### 確認すべきポイント

| 確認項目 | 確認方法 |
|---------|---------|
| 揚水ポンプは非常用電源に接続されているか | 電気系統図で確認 |
| 高架水槽の容量と枯渇までの時間 | 容量÷使用量で計算 |
| 非常電源未接続の場合の上階給水手段 | バケツリレー・ポンプ車等の検討 |

> **27施設のアンケートから**: 揚水ポンプの非常電源接続状況を即答できた施設は5施設。「施設課に聞かないと分からない」が大半の回答であった。

## 自動水栓・電動設備の停電対応

### 自動水栓が停電で使用不可になる問題

感染管理のために導入されたタッチレス（自動センサー式）水栓は、停電時にセンサーと電磁弁が動作せず、水が出なくなる。手動ハンドルがないため「ひねる」こともできない。

### 対策

1. **自動水栓の設置箇所リストの作成** — どの水栓が停電時に使えなくなるか把握
2. **手動水栓マップの作成** — 停電時に使える手動水栓の場所を全職員に周知
3. **アルコール消毒液の増備** — 流水手洗いの代替として十分な量を備蓄
4. **電動トイレの手動操作確認** — ウォシュレット等の停電時手動洗浄手順の周知

### チェックポイント

- [ ] 自動水栓の設置箇所と台数を把握しているか
- [ ] 停電時に手動で使える水栓の場所を周知しているか
- [ ] アルコール消毒液の備蓄が手洗い代替として十分か

> **27施設のアンケートから**: 自動水栓の停電時対応マニュアルを持つ施設は0施設であった。
