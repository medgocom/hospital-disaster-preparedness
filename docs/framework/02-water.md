# 軸2: 水道（断水対応）

## 概要

断水は医療行為の継続に重大な影響を与えます。手洗い、滅菌、透析、給食など、水は病院運営のあらゆる場面で必要です。

## リスク評価

### 水の使用用途と優先度

| 優先度 | 用途 | 1日使用量目安 | 代替手段 |
|-------|-----|-------------|---------|
| 最優先 | 飲料水 | 3L/人/日 | ペットボトル備蓄 |
| 最優先 | 透析用水 | 120L/回/人 | 給水車、井戸水RO処理 |
| 優先 | 手指消毒 | - | アルコール消毒 |
| 優先 | 医療器材洗浄 | - | ディスポ製品切替 |
| 通常 | トイレ洗浄 | - | 簡易トイレ |
| 通常 | 入浴 | - | 清拭対応 |
| 通常 | 給食調理 | - | 備蓄食・外部調達 |

### 自施設の給水体制

| 項目 | 内容 |
|-----|------|
| 受水槽容量 | m³（約　日分） |

## 対応フロー

### Phase 1: 初動（0〜30分）

```
断水発生
    ↓
受水槽残量確認
    ↓
節水指示発令
    ↓
優先使用エリア決定
```

### Phase 2: 節水対応（30分〜）

1. **即時節水措置**
   - トイレ洗浄の制限
   - シャワー・入浴の中止
   - 不要な水使用の禁止

2. **代替手段への切替**
   - 手洗い → アルコール消毒
   - 食器 → 使い捨て容器
   - 清拭 → ドライシャンプー等

3. **透析患者対応の決定**
   - 透析継続可否の判断
   - 転院手配（必要時）

### Phase 3: 外部調達（数時間〜）

- 給水車の要請（行政・自衛隊）
- ペットボトル水の追加調達
- 井戸水の活用検討

## 特殊考慮事項

### 透析施設向け

- RO水製造装置の電源確保
- 透析用水の備蓄または緊急調達ルート
- 近隣透析施設との連携協定

### 給食部門向け

- 備蓄食（水不要のもの）の確保
- 外部給食業者との災害時契約
- 経管栄養剤の備蓄
