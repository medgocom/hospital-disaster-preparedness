# 軸2: 水道（断水対応）

## 概要

断水は医療行為の継続に重大な影響を与えます。手洗い、滅菌、透析、給食など、水は病院運営のあらゆる場面で必要です。

## リスク評価

### 水の使用用途と優先度

| 優先度 | 用途 | 1日使用量目安 | 代替手段 |
|-------|-----|-------------|---------|
| 最優先 | 飲料水 | 3L/人/日 | ペットボトル備蓄 |
| 最優先 | 透析用水 | 120L/回/人 | 給水車、井戸水RO処理 |
| 優先 | 手指消毒 | - | アルコール消毒 |
| 優先 | 医療器材洗浄 | - | ディスポ製品切替 |
| 通常 | トイレ洗浄 | - | 簡易トイレ |
| 通常 | 入浴 | - | 清拭対応 |
| 通常 | 給食調理 | - | 備蓄食・外部調達 |

### 自施設の給水体制

| 項目 | 内容 |
|-----|------|
| 受水槽容量 | m³（約　日分） |
| 高架水槽容量 | m³ |
| 井戸設備 | あり / なし |
| 給水車受入設備 | あり / なし |
| 飲料水備蓄 | L（　日分） |

## 対応フロー

### Phase 1: 初動（0〜30分）

```
断水発生
    ↓
受水槽残量確認
    ↓
節水指示発令
    ↓
優先使用エリア決定
```

### Phase 2: 節水対応（30分〜）

1. **即時節水措置**
   - トイレ洗浄の制限
   - シャワー・入浴の中止
   - 不要な水使用の禁止

2. **代替手段への切替**
   - 手洗い → アルコール消毒
   - 食器 → 使い捨て容器
   - 清拭 → ドライシャンプー等

3. **透析患者対応の決定**
   - 透析継続可否の判断
   - 転院手配（必要時）

### Phase 3: 外部調達（数時間〜）

- 給水車の要請（行政・自衛隊）
- ペットボトル水の追加調達
- 井戸水の活用検討

## 特殊考慮事項

### 透析施設向け

- RO水製造装置の電源確保
- 透析用水の備蓄または緊急調達ルート
- 近隣透析施設との連携協定

### 給食部門向け

- 備蓄食（水不要のもの）の確保
- 外部給食業者との災害時契約
- 経管栄養剤の備蓄

## 実例と教訓（ケーススタディ）

### Case 1: 井水設備を持たない施設群の共通課題

**対象施設例**: 国立病院機構千葉医療センター、帝京大学医学部付属病院、東北労災病院、仙台オープン病院、久留米大学医療センター、済生会横浜市東部病院 など

**リスク**:
- 上水道が途絶した場合、**代替水源がない**
- 透析、手術、滅菌など水を大量に使用する医療行為が制限される
- 飲料水・生活用水の確保も困難になる

**推奨対策**:

| 期間 | 対策 |
|-----|------|
| **短期** | 給水車の手配先リスト作成（複数業者）<br>ペットボトル水の備蓄（最低3日分、可能なら7日分）<br>近隣の井戸保有施設との協定 |
| **中期** | 貯水槽の増設または大型化<br>雨水貯留システムの導入（トイレ洗浄用など） |
| **長期** | 井水設備の新設検討<br>地下水脈調査の実施 |

**タグ**: `#井水なし` `#断水脆弱` `#代替水源なし`

---

### Case 2: 東北労災病院（貯水制限）

**状況**: 上水貯留が通常時の50%×3日分と制限付き

**リスク**:
- 災害時に使用量が増加した場合、3日持たない可能性
- 復旧が長期化した場合の対応が困難

**教訓**:
1. 貯水量は「**通常時の100%×3日分**」を目標とする
2. 災害時の節水プロトコルを事前に策定
3. 優先度の高い用途（透析、手術、飲料）と低い用途（清掃、洗濯）を明確化

**タグ**: `#貯水制限`
