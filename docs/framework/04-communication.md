# 軸4: 通信（通信途絶対応）

## 概要

災害時の通信途絶は、院内の情報共有と外部との連絡を困難にします。固定電話、携帯電話、インターネットが使用不能になった場合の代替手段を準備しておく必要があります。

## リスク評価

### 通信手段と途絶リスク

| 通信手段 | 途絶リスク | 途絶時の影響 | 代替手段 |
|---------|----------|-------------|---------|
| 固定電話 | 地震・水害で断線 | 外部連絡不可 | 携帯電話 |
| 携帯電話 | 基地局停止・輻輳 | 外部連絡不可 | 衛星電話 |
| インターネット | 回線断・停電 | メール・Web不可 | 衛星通信 |
| 院内PHS | 基地局停電 | 院内連絡不可 | トランシーバー |
| 院内放送 | 停電・設備故障 | 一斉周知不可 | メガホン |

### 自施設の通信体制

| 項目 | 内容 |
|-----|------|
| 衛星電話 | あり / なし（台数：　） |
| 無線機 | あり / なし（台数：　） |
| MCA無線 | あり / なし |
| 防災無線 | あり / なし |
| 非常用電話回線 | あり / なし |

## 対応フロー

### Phase 1: 初動（0〜15分）

```
通信障害発生
    ↓
使用可能な通信手段の確認
    ↓
[衛星電話あり]        [通信手段なし]
    ↓                    ↓
外部連絡開始        伝令体制構築
    ↓                    ↓
院内はトランシーバー  人による情報伝達
```

### Phase 2: 院内情報伝達体制（15分〜）

1. **伝令体制の構築**
   - 各部署に伝令担当者配置
   - 定時巡回ルートの設定
   - 災害対策本部への報告経路

2. **掲示板・ホワイトボードの活用**
   - 災害対策本部前に情報掲示
   - 各部署に状況ボード設置

3. **メガホン・拡声器の配備**
   - 院内放送代替として使用

### Phase 3: 外部連絡

- 衛星電話による本部連絡
- 防災無線による行政連絡
- 使者派遣（最終手段）

## 通信訓練の重要事項

### トランシーバー使用ルール

1. 送信前に「〇〇から△△へ」と呼びかけ
2. 応答を待ってから本題
3. 終話時は「了解、どうぞ」で締め
4. 短く簡潔に（30秒以内）

### 衛星電話使用上の注意

- 屋外・空が見える場所で使用
- 接続まで時間がかかる場合あり
- 通話料金が高額（要登録管理）

## SNS（LINE等）の活用

### アンケートで挙がった声

> 「LINEなどのSNS連絡網の構築」
> 「電話以外の連絡網」

### LINE連絡網の構築手順

1. **グループ作成**
   - 【災害対策本部】（幹部用）
   - 【○○病棟】（部署別）
   - 【全職員】（一斉周知用）

2. **運用ルール**
   - 災害時のみ使用（日常連絡禁止）
   - 既読確認＝安否確認として扱う
   - 写真・動画での状況共有を推奨

3. **注意事項**
   - 個人情報の取扱い（患者情報は送らない）
   - 停電時はバッテリー節約のため最小限に
   - 通信輻輳時はテキストのみ（画像送信回避）

### LINE連絡網テンプレート

```
【災害発生連絡】
発生日時：○月○日 ○時○分
事象：（地震/停電/その他）
病院状況：（被害なし/軽微/重大）
出勤要請：（あり/なし/待機）
返信：「確認」と返信してください
```

## 中間ネットワーク機器の電源問題

### PHS基地局・ネットワークハブの盲点

院内通信が途絶する原因は「基幹設備の停止」だけではない。PHS基地局、ネットワークスイッチ（ハブ）、Wi-Fiアクセスポイントなど、**中間機器の電源喪失**が院内通信を麻痺させる。

| 機器 | 設置場所 | 非常電源接続 | 停止時の影響 |
|-----|---------|------------|-------------|
| PHS基地局 | 各フロア天井裏 | 多くの施設で未接続 | 院内PHSが全滅 |
| L2/L3スイッチ | EPS（電気シャフト） | 多くの施設で未接続 | 電子カルテ端末が孤立 |
| Wi-Fiアクセスポイント | 各フロア | ほぼ未接続 | 無線端末が使用不可 |
| PBX（構内交換機） | サーバー室/通信室 | 接続済みが多い | 院内外の電話が全滅 |

### 確認すべきポイント

1. PHS基地局が非常用電源に接続されているか
2. 各フロアのネットワークスイッチにUPSが設置されているか
3. PBXの冷却が停電時も維持されるか（夏季の熱暴走リスク）
4. Wi-Fiが使えない場合の代替手段（トランシーバー・伝令）が準備されているか

### チェックポイント

- [ ] PHS基地局の非常電源接続状況を把握しているか
- [ ] ネットワークスイッチ（ハブ）の電源系統を確認しているか
- [ ] PBX室の空調が非常用電源に接続されているか

> **27施設のアンケートから**: PHS基地局の非常電源接続状況を把握している施設は3割未満。「考えたことがなかった」が最多回答。

## トランシーバー運用の詳細

### 機種選定のポイント

| 種類 | 特徴 | 推奨場面 |
|-----|-----|---------|
| 特定小電力無線 | 免許不要、院内向け | 院内連絡 |
| 簡易無線 | 登録制、広範囲 | 敷地内・近隣 |
| IP無線 | 携帯回線利用 | 広域連携 |

### チャンネル割当例

| チャンネル | 用途 | 使用者 |
|-----------|-----|-------|
| CH1 | 災害対策本部 | 本部要員 |
| CH2 | 医療班 | 医師・看護師 |
| CH3 | ロジ班 | 事務・施設 |
| CH4 | 予備 | 緊急時 |

### 通話プロトコル

```
呼出：「本部から医療班、本部から医療班、応答願います」
応答：「医療班です、どうぞ」
本題：「救急外来に傷病者10名到着予定、受入準備願います」
確認：「了解、救急外来で受入準備します」
終話：「本部了解、以上」
```

## 実例と教訓（ケーススタディ）

### Case 1: 複合災害時のアクセス寸断

**対象施設例**: 新ひだか町立静内病院

**状況**: 津波、土砂災害、洪水のハザードエリア内に位置

**リスク**:
- 複合災害発生時、物理的なアクセスが寸断される可能性
- 通信インフラ（電話回線、携帯基地局）も同時に被災
- **孤立状態**で外部との連絡が取れなくなる

**推奨対策**:
1. **衛星電話の配備**（最低1台、理想は複数台）
2. アマチュア無線局の設置と運用者の確保
3. ドローンによる通信中継の検討
4. 近隣施設とのトランシーバー通信網の構築

**タグ**: `#複合災害` `#アクセス寸断` `#孤立`

## 夜間・休日対応（少人数での初動）

夜間の通信途絶は、外部への支援要請と院内連絡の両方を困難にする。特に院内PHSが使えなくなると、広い院内での情報伝達手段が極めて限られる。

### 通信途絶発見時の初動フロー（当直者向け）

```
院内PHS/電話が通じないことを確認
  ↓
❶ 複数手段で確認（固定電話・PHS・携帯電話）
  ↓
❷ 停電が原因か確認（停電→PHS基地局停止の可能性）
  ↓
  ├─ [停電あり] → 電気系統の復旧を優先（→軸1参照）
  │
  └─ [停電なし] → 通信設備自体の故障 or 外部回線断
                   → 代替通信手段の確保へ
  ↓
❸ 災害BOXからトランシーバーを取り出し、本部に報告
```

### 夜勤者の役割分担

| 役割 | 担当 | やること |
|------|------|---------|
| 指揮・判断 | 当直医師 | 院外連絡の要否判断、伝令体制の指示 |
| 院内連絡 | 夜勤看護師 | 各病棟への直接伝達（巡回伝令） |
| 外部連絡 | 当直事務 | 衛星電話での外部連絡試行 |

### 夜間通信途絶時の3大チェック

1. **衛星電話は使えるか**（保管場所から取り出し、屋外で起動確認）
2. **トランシーバーは充電されているか**（災害BOXから取り出し動作確認）
3. **携帯電話のデータ通信は生きているか**（LINEでの連絡が可能か確認）

### 夜間の最低限連絡体制

通信手段が限られる中で、以下の優先順位で連絡を確保する。

| 優先度 | 連絡先 | 手段（優先順） |
|-------|-------|-------------|
| 1 | 院内各病棟 | トランシーバー → 伝令（直接巡回） |
| 2 | 院長・副院長 | 携帯電話 → ショートメール → LINE |
| 3 | 消防・行政 | 衛星電話 → 防災無線 |
| 4 | 参集対象職員 | LINE一斉送信 → 電話（順次） |

### 伝令による院内情報伝達

通信手段がすべて使えない場合の最終手段。

```
【夜間伝令ルート】
当直医師（指揮所=ナースステーション）
  → 3F病棟 → 4F病棟 → 5F病棟
  → 2F手術室/ICU
  → 1F救急外来/事務室
  → 地下（設備室）
  → 指揮所に戻り報告
```

> **事前準備**: 衛星電話の保管場所・操作方法・暗証番号を当直者全員が把握していること。トランシーバーは災害BOXに常時充電状態で保管すること。
