# 軸4: 通信（通信途絶対応）

## 概要

災害時の通信途絶は、院内の情報共有と外部との連絡を困難にします。固定電話、携帯電話、インターネットが使用不能になった場合の代替手段を準備しておく必要があります。

## リスク評価

### 通信手段と途絶リスク

| 通信手段 | 途絶リスク | 途絶時の影響 | 代替手段 |
|---------|----------|-------------|---------|
| 固定電話 | 地震・水害で断線 | 外部連絡不可 | 携帯電話 |
| 携帯電話 | 基地局停止・輻輳 | 外部連絡不可 | 衛星電話 |
| インターネット | 回線断・停電 | メール・Web不可 | 衛星通信 |
| 院内PHS | 基地局停電 | 院内連絡不可 | トランシーバー |
| 院内放送 | 停電・設備故障 | 一斉周知不可 | メガホン |

## 対応フロー

### Phase 1: 初動（0〜15分）

```
通信障害発生
    ↓
使用可能な通信手段の確認
    ↓
[衛星電話あり]        [通信手段なし]
    ↓                    ↓
外部連絡開始        伝令体制構築
    ↓                    ↓
院内はトランシーバー  人による情報伝達
```

### Phase 2: 院内情報伝達体制（15分〜）

1. **伝令体制の構築**
   - 各部署に伝令担当者配置
   - 定時巡回ルートの設定
   - 災害対策本部への報告経路

2. **掲示板・ホワイトボードの活用**
   - 災害対策本部前に情報掲示
   - 各部署に状況ボード設置

3. **メガホン・拡声器の配備**
   - 院内放送代替として使用

### Phase 3: 外部連絡

- 衛星電話による本部連絡
- 防災無線による行政連絡
- 使者派遣（最終手段）

## 災害時優先電話

### 災害時優先電話とは

NTTの災害時優先電話として登録された回線は、輻輳時でも優先的に接続されます。

- 登録状況の確認
- 設置場所の周知
- 使用ルールの明確化

## 通信訓練の重要事項

### トランシーバー使用ルール

1. 送信前に「〇〇から△△へ」と呼びかけ
2. 応答を待ってから本題
3. 終話時は「了解、どうぞ」で締め
4. 短く簡潔に（30秒以内）

### 衛星電話使用上の注意

- 屋外・空が見える場所で使用
- 接続まで時間がかかる場合あり
- 通話料金が高額（要登録管理）
