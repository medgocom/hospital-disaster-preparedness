# 軸7: 空調（HVAC停止対応）

## 概要

空調停止は、手術室・ICUの陽圧管理、薬品・検体の温度管理、患者の体温管理など、多岐にわたる影響を及ぼします。特に夏季の高温・冬季の低温は患者の生命にかかわります。

## リスク評価

### 空調停止の影響

| エリア | 影響 | 許容時間 | 対応策 |
|-------|-----|---------|-------|
| 手術室 | 清浄度低下・感染リスク | 数時間 | 手術延期・移送 |
| ICU/CCU | 温度管理困難 | 数時間 | 携帯冷暖房 |
| 薬局 | 薬品温度逸脱 | 1〜2時間 | 保冷剤・冷蔵庫移動 |
| 検査室 | 検体・試薬劣化 | 1〜2時間 | 検査制限 |
| NICU | 保育器依存 | - | 保育器で対応可 |
| 一般病棟 | 患者不快・体調悪化 | 数時間〜 | 窓開放・扇風機 |

## 対応フロー

### Phase 1: 初動（0〜30分）

```
空調停止
    ↓
原因確認（停電 or 設備故障）
    ↓
優先エリアの温度監視開始
    ↓
窓開放 or 閉鎖の判断
```

### Phase 2: 応急対応（30分〜）

#### 夏季（高温時）

1. **換気の確保**
   - 窓の開放（可能な場所）
   - 扇風機の配備

2. **患者ケア**
   - 氷枕・冷却シートの使用
   - 飲水の励行
   - 衣類の調整
   - 高リスク患者の監視強化

3. **薬品・検体の保護**
   - 保冷剤の活用
   - 冷蔵が必要な薬品の移動
   - 検査の制限

#### 冬季（低温時）

1. **保温の確保**
   - 毛布の追加配布
   - 電気毛布（電源あれば）
   - 湯たんぽ

2. **窓・ドアの管理**
   - 開放を最小限に
   - カーテン活用

3. **高リスク患者**
   - 新生児
   - 高齢者
   - 低体温リスク患者

### Phase 3: 継続対応

- 携帯型冷暖房機器の調達
- 患者転院の検討
- 手術スケジュールの見直し

## 温度管理が必要なエリア・物品

### エリア別目標温度

| エリア | 目標温度 | 許容範囲 | 備考 |
|-------|---------|---------|------|
| 手術室 | 22〜26℃ | ±2℃ | 術野・患者体温考慮 |
| ICU | 24〜26℃ | ±2℃ | |
| NICU | 26〜28℃ | ±1℃ | 保育器で個別管理 |
| 薬局 | 25℃以下 | - | 薬品保管条件 |
| 血液保管 | 2〜6℃ | - | 冷蔵庫管理 |
| 一般病棟 | 24〜27℃ | ±3℃ | |

### 温度管理が必要な物品

| 物品 | 保管温度 | 逸脱時の影響 |
|-----|---------|-------------|
| インスリン | 2〜8℃ | 効力低下 |
| ワクチン | 2〜8℃ | 失活 |
| 血液製剤 | 製剤による | 使用不可 |
| 検査試薬 | 製品による | 精度低下 |
