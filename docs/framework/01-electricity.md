# 軸1: 電気（停電対応）

## 概要

停電は病院機能に最も深刻な影響を与えるインフラ途絶です。人工呼吸器、透析装置、手術室機器など、生命維持に直結する医療機器の多くが電力に依存しています。

## リスク評価

### 影響を受ける主要設備・機能

| カテゴリ | 設備・機能 | 影響度 | 備考 |
|---------|----------|-------|------|
| 生命維持 | 人工呼吸器 | 致命的 | 内部バッテリー30分〜2時間 |
| 生命維持 | 透析装置 | 致命的 | |
| 診療 | 電子カルテ | 重大 | UPS対応時間を確認 |
| 診療 | 医療画像システム | 重大 | |
| 設備 | エレベーター | 重大 | 患者搬送不可 |
| 設備 | 空調 | 重大 | 温度管理必要エリアあり |
| 照明 | 一般照明 | 中 | 非常灯で対応 |

## 対応フロー

### Phase 1: 初動（0〜10分）

```
停電発生
    ↓
非常用発電機 自動起動確認
    ↓
[起動成功]           [起動失敗]
    ↓                    ↓
給電範囲確認      手動起動試行
    ↓                    ↓
患者安全確認      外部支援要請
```

### Phase 2: 安定化（10分〜1時間）

1. **電力使用制限の発令**
   - 不要な機器の電源OFF
   - 空調設定温度の緩和

2. **優先給電エリアの確認**
   - ICU/CCU
   - 手術室（進行中手術）
   - NICU
   - 透析室

3. **燃料残量の確認と補給手配**

### Phase 3: 継続対応（1時間〜）

- 長期化の判断基準設定
- 患者転院の検討開始
- 職員シフト体制の調整
- 外部機関への状況報告

## 地域別の追加対策

### 寒冷地（北海道等）：ブラックアウト対策

2018年北海道胆振東部地震では、全道295万戸が停電する「ブラックアウト」が発生しました。寒冷地では停電＝暖房停止であり、患者の生命に直結します。

#### 札幌東徳洲会病院の教訓

> 「コージェネレーションシステムが稼働しなかった」
> 「非常用コンセント（赤コンセント）が使えなかった」

#### 寒冷地特有のチェック項目

- [ ] コージェネレーションシステムの定期点検
- [ ] 非常用コンセント（赤コンセント）の動作確認
- [ ] 暖房用燃料の備蓄量（厳冬期想定）
- [ ] 電気毛布・湯たんぽ等の代替暖房手段
- [ ] 凍結防止対策（配管ヒーター等）

#### 冬季停電時の優先事項

1. **暖房の確保**（最優先）
   - ボイラー燃料の確認
   - 電気暖房→灯油ストーブ等への切替

2. **凍結防止**
   - 給水管の凍結防止
   - 医療機器の温度管理

3. **患者の保温**
   - 毛布の追加配布
   - 高リスク患者（高齢者・新生児）の集約

### 離島：外部支援到達までの自己完結

離島では停電復旧に時間がかかる場合があります。

#### 離島特有のチェック項目

- [ ] 発電機燃料の備蓄量（1週間分以上推奨）
- [ ] 燃料の海上輸送手配ルート
- [ ] 太陽光・蓄電池等の代替電源
