# 軸1: 電気（停電対応）

## 概要

停電は病院機能に最も深刻な影響を与えるインフラ途絶です。人工呼吸器、透析装置、手術室機器など、生命維持に直結する医療機器の多くが電力に依存しています。

## リスク評価

### 影響を受ける主要設備・機能

| カテゴリ | 設備・機能 | 影響度 | 備考 |
|---------|----------|-------|------|
| 生命維持 | 人工呼吸器 | 致命的 | 内部バッテリー30分〜2時間 |
| 生命維持 | 透析装置 | 致命的 | |
| 診療 | 電子カルテ | 重大 | UPS対応時間を確認 |
| 診療 | 医療画像システム | 重大 | |
| 設備 | エレベーター | 重大 | 患者搬送不可 |
| 設備 | 空調 | 重大 | 温度管理必要エリアあり |
| 照明 | 一般照明 | 中 | 非常灯で対応 |

### 自施設の非常用電源体制

| 項目 | 内容 |

| 非常用発電機 | 台数：　　容量：　　kVA |

## 対応フロー

### Phase 1: 初動（0〜10分）

```
停電発生
    ↓
非常用発電機 自動起動確認
    ↓
[起動成功]           [起動失敗]
    ↓                    ↓
給電範囲確認      手動起動試行
    ↓                    ↓
患者安全確認      外部支援要請
```

### Phase 2: 安定化（10分〜1時間）

1. **電力使用制限の発令**
   - 不要な機器の電源OFF
   - 空調設定温度の緩和

2. **優先給電エリアの確認**
   - ICU/CCU
   - 手術室（進行中手術）
   - NICU
   - 透析室

3. **燃料残量の確認と補給手配**

### Phase 3: 継続対応（1時間〜）

- 長期化の判断基準設定
- 患者転院の検討開始
- 職員シフト体制の調整
- 外部機関への状況報告

## 地域別の追加対策

### 寒冷地（北海道等）：ブラックアウト対策

2018年北海道胆振東部地震では、全道295万戸が停電する「ブラックアウト」が発生しました。寒冷地では停電＝暖房停止であり、患者の生命に直結します。

#### 札幌東徳洲会病院の教訓

> 「コージェネレーションシステムが稼働しなかった」
> 「非常用コンセント（赤コンセント）が使えなかった」

#### 寒冷地特有のチェック項目

- [ ] コージェネレーションシステムの定期点検
- [ ] 非常用コンセント（赤コンセント）の動作確認
- [ ] 暖房用燃料の備蓄量（厳冬期想定）
- [ ] 電気毛布・湯たんぽ等の代替暖房手段
- [ ] 凍結防止対策（配管ヒーター等）

#### 冬季停電時の優先事項

1. **暖房の確保**（最優先）
   - ボイラー燃料の確認
   - 電気暖房→灯油ストーブ等への切替

2. **凍結防止**
   - 給水管の凍結防止
   - 医療機器の温度管理

3. **患者の保温**
   - 毛布の追加配布
   - 高リスク患者（高齢者・新生児）の集約

### 離島：外部支援到達までの自己完結

離島では停電復旧に時間がかかる場合があります。

#### 離島特有のチェック項目

- [ ] 発電機燃料の備蓄量（1週間分以上推奨）
- [ ] 燃料の海上輸送手配ルート
- [ ] 太陽光・蓄電池等の代替電源

## 実例と教訓（ケーススタディ）

### Case 1: 熊本赤十字病院（2016年熊本地震）

**状況**: 建物自体は免震構造で無事だったが、高圧配電ケーブルが地震の揺れで故障

**結果**:
- 商用電源が使用不能
- 自家発電機への切り替えを試みるも、配電盤までの経路が同じケーブルだったため自家発電も使用不能
- **完全な「電源喪失」状態**に陥った

**教訓**:
1. 商用電源と非常用電源の配電経路を**物理的に分離**する
2. 配電ケーブルの耐震対策（フレキシブル接続など）を施す
3. 電源経路の単一障害点（SPOF）を特定し、冗長化する

**タグ**: `#電源単一障害` `#ケーブル故障` `#自家発不能`

---

### Case 2: 田主丸中央病院（水害事例）

**状況**: 河川氾濫により1階が浸水

**結果**:
- 1階に設置されていた電気設備が水没
- 揚水ポンプも水没し、停電と断水が同時発生
- 病院機能が大幅に制限された

**教訓**:
1. 電気設備（受変電設備、配電盤）は**2階以上**または防水対策を講じた場所に設置
2. ハザードマップで浸水想定区域を確認し、設備配置を見直す
3. 止水板、防水扉などの浸水対策を整備する

**タグ**: `#電源水没` `#1階設備` `#水害リスク`

---

### Case 3: 東海大学医学部付属病院（設計上の課題）

**状況**: 電力引き込みが1回線のみ（多くの病院は2回線以上）

**リスク**:
- 電力会社からの引き込み経路に障害が発生した場合、商用電源を完全に喪失
- 自家発電のみに依存する状態となり、燃料切れのリスクが増大

**推奨対策**:
1. 可能であれば2回線引き込み（異なる変電所から）に改修
2. 困難な場合は、燃料備蓄の増強と燃料調達協定の締結
3. 近隣施設との電力融通協定の検討

**タグ**: `#電源単一回線` `#冗長性不足`
