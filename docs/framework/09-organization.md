# 軸9: 組織体制（指揮命令系統・多職種連携）

## 概要

アンケート結果から、インフラのハード面だけでなく「組織としてどう動くか」への関心が非常に高いことが判明しました。本軸では、災害時の指揮命令系統、役割分担、多職種連携について整理します。

## リスク評価

### 組織体制の崩壊がもたらす影響

| 状況 | 影響 | 対応策 |
|-----|-----|-------|
| 指揮者不在 | 意思決定停滞 | 代行順位の明確化 |
| 役割不明確 | 重複・漏れ発生 | アクションカード |
| 情報途絶 | 状況把握不能 | クロノロジー・EMIS |
| 連携不足 | 患者対応遅延 | 多職種訓練 |

## 災害時の組織体制

### 災害対策本部の構成

```
┌─────────────────────────────────────────────┐
│              災害対策本部長                  │
│            （院長 → 副院長）                │
└─────────────────┬───────────────────────────┘
                  │
    ┌─────────────┼─────────────┐
    ▼             ▼             ▼
┌───────┐   ┌───────┐   ┌───────┐
│情報班 │   │医療班 │   │ロジ班│
└───────┘   └───────┘   └───────┘
・クロノロ   ・トリアージ  ・物資管理
・EMIS入力   ・治療調整    ・人員配置
・外部連絡   ・転院調整    ・設備管理
```

### 代行順位の明確化

| 役職 | 第1代行 | 第2代行 | 第3代行 |
|-----|--------|--------|--------|
| 本部長（院長） | 副院長 | 診療部長 | 当直医師 |
| 情報班長 | 事務部長 | 医事課長 | 当直事務 |
| 医療班長 | 看護部長 | 副看護部長 | 当直師長 |
| ロジ班長 | 事務次長 | 施設課長 | 当直事務 |

## 多職種連携のルール

### アンケートで挙がった課題

> 「誰がトリアージするのか」
> 「PT（理学療法士）でもトリアージの現場でトリアージできるのか」
> 「トリアージ時の連携」

### 職種別の役割定義

| 職種 | 災害時の主な役割 | トリアージ可否 |
|-----|----------------|--------------|
| 医師 | 治療判断、トリアージ統括 | ◎ 実施・最終判断 |
| 看護師 | トリアージ実施、患者ケア | ◎ 実施可能 |
| 救急救命士 | トリアージ実施、搬送 | ◎ 実施可能 |
| 薬剤師 | 医薬品管理、調剤 | △ 補助のみ |
| 臨床検査技師 | 検体検査、輸血対応 | △ 補助のみ |
| 放射線技師 | 画像検査 | △ 補助のみ |
| 理学療法士 | 搬送補助、リハ評価 | △ 研修修了者は可 |
| 事務職員 | 受付、情報管理、記録 | × 不可 |

### トリアージ実施のルール

1. **基本**: 医師・看護師・救急救命士が実施
2. **例外**: 研修を修了したコメディカルは「一次トリアージ」のみ可
3. **最終判断**: 必ず医師が確認・承認
4. **記録**: トリアージタグ + クロノロジーに記載

## 情報共有ツール

### クロノロジー（時系列記録）

| 時刻 | 発信者 | 内容 | 対応状況 |
|-----|-------|-----|---------|
| 09:00 | 当直医 | 震度5強、停電発生 | 本部設置 |
| 09:05 | 施設課 | 非常電源起動確認 | 完了 |
| 09:10 | 病棟A | 人工呼吸器患者3名安全確認 | 完了 |
| 09:15 | 救急 | 傷病者10名来院予定 | 対応中 |

### EMIS（広域災害救急医療情報システム）

- **入力タイミング**: 災害発生後30分以内
- **入力担当者**: 情報班（事務職員）
- **入力内容**:
  - 病院の被災状況
  - 受入可能患者数
  - 必要な支援

### 情報伝達のルール

```
【報告の5W1H】
When:   いつ発生したか
Where:  どこで発生したか
What:   何が起きたか
Who:    誰が関係しているか
Why:    なぜ起きたか（分かれば）
How:    現在の状況・対応状況
```

## 夜間・休日体制

### 課題認識

> 「夜間・休日（幹部不在時）の組織図作成」が必要

### 夜間・休日の組織図

```
【当直体制】
┌─────────────────────────────┐
│   当直医師（最高意思決定者）   │
└───────────┬─────────────────┘
            │
    ┌───────┴───────┐
    ▼               ▼
┌───────┐     ┌───────┐
│夜勤師長│     │当直事務│
└───────┘     └───────┘
    │               │
    ▼               ▼
各病棟夜勤者    施設管理当直
```

### 連絡基準

| 状況 | 連絡先 | 連絡手段 |
|-----|-------|---------|
| レベル1（注意） | 管理当直のみ | 翌朝報告でも可 |
| レベル2（警戒） | 院長・副院長 | 電話 |
| レベル3（災害） | 全幹部 | 緊急連絡網 |

## 訓練ポイント

1. **机上訓練（図上訓練）**
   - 災害シナリオに基づく意思決定訓練
   - 各班の連携確認

2. **実動訓練**
   - 災害対策本部の設置訓練
   - クロノロジー記入訓練
   - EMIS入力訓練

3. **夜間想定訓練**
   - 当直体制での発災対応
   - 幹部への連絡訓練
