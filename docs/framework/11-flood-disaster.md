# 軸11: 水害対策（浸水・洪水対応）

## 概要

近年、気候変動の影響により水害リスクが増大しています。河川流域や低地に位置する病院は、浸水・洪水による被害を想定した対策が必要です。

## リスク評価

### 水害の種類と影響

| 水害の種類 | 原因 | 影響 |
|-----------|-----|------|
| 河川氾濫 | 堤防決壊、越水 | 広範囲の浸水、長期化 |
| 内水氾濫 | 排水能力超過 | 局所的な浸水 |
| 高潮 | 台風、低気圧 | 沿岸部の浸水 |
| 土砂災害 | 大雨、地震 | 建物損壊、アクセス寸断 |

### 浸水深と被害の関係

| 浸水深 | 被害レベル | 対応 |
|-------|----------|------|
| 〜0.5m | 床下浸水 | 土嚢、止水板で防御可能 |
| 0.5〜1m | 床上浸水 | 1階機能に影響 |
| 1〜2m | 1階水没 | 1階機能喪失、垂直避難 |
| 2〜5m | 複数階水没 | 病院機能の大部分喪失 |
| 5m〜 | 建物水没 | 全機能喪失、救助待ち |

## 実例と教訓（ケーススタディ）

### Case 1: まび記念病院（2018年西日本豪雨）

**状況**: 岡山県倉敷市真備町の河川流域・低地に位置

**結果**:
- 堤防決壊により病院周辺が水没（**最大4〜5m**）
- 1階機能が完全に喪失
- 病院が**孤立**し、患者・職員が上階に避難
- **ヘリコプターによる救出**が必要となった

**教訓**:
1. ハザードマップの確認と浸水想定深さの把握
2. 重要設備・機能の**上階への配置**
3. 垂直避難計画の策定（何階まで、何人が、どのルートで）
4. 孤立時の備蓄（食料、水、医薬品、燃料）の確保
5. 救出要請手段の確保（衛星電話、ヘリポート）

**タグ**: `#水害リスク` `#水没` `#1階全喪失` `#孤立` `#避難困難`

---

### Case 2: 田主丸中央病院（水害事例）

**状況**: 福岡県久留米市、河川流域に位置

**結果**:
- 河川氾濫により1階が浸水
- 1階に設置されていた**電気設備・揚水ポンプが水没**
- 停電と断水が同時発生

**教訓**:
1. 電気設備（受変電設備、配電盤）は**2階以上**に設置
2. 揚水ポンプ・給水設備の浸水対策
3. 止水板、防水扉の整備

**タグ**: `#水害リスク` `#電源水没` `#断水リスク` `#1階設備`

---

### Case 3: 水害対策が不明確な施設群

**対象施設例**: 済生会江津総合病院、市立福知山市民病院

**リスク**:
- 水害対策に関する記載がなく、備えの状況が不明
- 河川流域に位置する施設は浸水リスクが高い

**推奨対策**:
1. 自施設のハザードマップを確認
2. 浸水想定深さに応じた対策レベルを設定
3. 止水板・土嚢の備蓄
4. 排水ポンプの配備
5. 重要書類・データのバックアップ（上階またはクラウド）

**タグ**: `#水害対策不明` `#河川流域`

## 対応フロー

### 水害警戒時

```
大雨・洪水警報発令
    ↓
ハザードマップ確認
    ↓
[浸水想定区域外]      [浸水想定区域内]
    ↓                    ↓
通常警戒            水害対策準備
                       ↓
                  止水板設置
                  土嚢配置
                  排水ポンプ準備
                       ↓
                  垂直避難準備
                  1階機能の上階移転
```

### 浸水発生時

```
浸水開始
    ↓
1階からの避難
    ↓
電気設備の緊急停止
    ↓
垂直避難の実施
    ↓
[孤立]              [避難路確保]
    ↓                    ↓
救助要請            院外避難検討
（衛星電話・ヘリ）
```

## 事前対策

### ハード対策

| 対策 | 内容 | コスト |
|-----|------|-------|
| 止水板 | 出入口への設置 | 中 |
| 防水扉 | 地下・1階への導入 | 高 |
| 土嚢 | 備蓄（砂入り/水で膨らむタイプ） | 低 |
| 排水ポンプ | 浸水時の排水 | 中 |
| 重要設備の上階移設 | 受変電設備、サーバー等 | 高 |

### ソフト対策

| 対策 | 内容 |
|-----|------|
| ハザードマップ確認 | 浸水想定深さ、継続時間の把握 |
| 垂直避難計画 | 避難先階、避難経路、搬送方法 |
| 孤立時備蓄 | 食料、水、医薬品（想定孤立日数分） |
| 救出手段の確保 | 衛星電話、ヘリポート指定 |
| BCP見直し | 水害シナリオの追加 |

## 垂直避難計画

### 避難先階の決定

浸水想定深さ + 余裕（1階分）以上の階へ避難

| 浸水想定 | 避難先階 |
|---------|---------|
| 〜2m | 2階以上 |
| 2〜5m | 3階以上 |
| 5m〜 | 最上階 |

### 避難対象の優先順位

1. 自力歩行困難な患者（ICU、術後、高齢者）
2. 医療機器依存患者（人工呼吸器、透析）
3. 一般入院患者
4. 外来患者
5. 職員

### 搬送に必要な人員

| 患者状態 | 必要人員 |
|---------|---------|
| 自力歩行可能 | 1名（誘導） |
| 車椅子 | 2名 |
| ストレッチャー | 4名 |
| 人工呼吸器装着 | 4名+機器担当1名 |
