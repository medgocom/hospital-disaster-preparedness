# 水害対策に関するBCP対策

## 概要

河川流域や低地に位置する病院は、水害による浸水リスクへの備えが不可欠である。

## リスク評価

### 浸水リスクの確認

| 確認項目 | 確認方法 |
|---------|---------|
| ハザードマップ | 市区町村のWebサイト、国土交通省ハザードマップポータル |
| 浸水想定深さ | ハザードマップで確認（0.5m未満〜5m以上） |
| 浸水継続時間 | 想定される浸水の継続時間 |
| 過去の浸水履歴 | 地域の災害記録、古老への聞き取り |

### 浸水深さ別の影響

| 浸水深さ | 影響 | 対策レベル |
|---------|-----|-----------|
| 0.5m未満 | 床上浸水、1階機能制限 | 止水板で対応可能 |
| 0.5〜1m | 1階設備水没リスク | 重要設備の移設検討 |
| 1〜2m | 1階機能喪失 | 垂直避難計画必須 |
| 2m以上 | 2階以上への影響 | 病院避難も検討 |

### 自施設の水害リスクチェック

| 項目 | 内容 |
|-----|------|
| ハザードマップ確認 | 済 / 未確認 |
| 浸水想定深さ | m |
| 1階に配置された重要設備 | |
| 止水板・土嚢の備蓄 | あり / なし |
| 垂直避難計画 | 策定済 / 未策定 |

## 対応フロー

### Phase 1: 警戒段階（大雨警報発令時）

```
大雨警報発令
    ↓
河川水位の監視開始
    ↓
止水板・土嚢の準備
    ↓
1階の重要物資を上階へ移動
    ↓
避難準備情報の確認
```

### Phase 2: 浸水開始時

1. **垂直避難の実施**
   - 患者を上階へ移動
   - 歩行困難患者の優先搬送

2. **電気設備の保護**
   - 1階分電盤の遮断（感電防止）
   - 非常用発電機の確認

3. **情報収集・発信**
   - 本部への状況報告
   - 救援要請の準備

### Phase 3: 浸水中・孤立時

- 備蓄物資での継続運営
- 外部への救援要請（衛星電話等）
- ヘリコプター救助の要請（必要時）
- 水位低下まで待機

## 必要備品チェックリスト

- [ ] 止水板（出入口用）
- [ ] 土嚢（予備）
- [ ] 排水ポンプ
- [ ] 長靴・レインコート
- [ ] ライフジャケット（浸水深が深い場合）
- [ ] ボート（孤立リスクが高い場合）
- [ ] 衛星電話（孤立時の通信用）
- [ ] 上階への備蓄物資（食料・水・医薬品）

## 訓練ポイント

1. 止水板設置訓練
2. 垂直避難訓練（患者搬送含む）
3. 1階機能喪失時の代替運用訓練
4. 孤立時の通信訓練

## 実例と教訓（ケーススタディ）

### Case 1: まび記念病院（2018年西日本豪雨）

**状況**: 岡山県倉敷市真備町の河川流域・低地に位置

**結果**:
- 堤防決壊により病院周辺が水没（最大4〜5m）
- 1階機能が完全に喪失
- 病院が孤立し、患者・職員が上階に避難
- ヘリコプターによる救出が必要となった

**教訓**:
1. ハザードマップの確認と浸水想定深さの把握
2. 重要設備・機能の上階への配置
3. 垂直避難計画の策定（何階まで、何人が、どのルートで）
4. 孤立時の備蓄（食料、水、医薬品、燃料）の確保
5. 救出要請手段の確保（衛星電話、ヘリポート）

**タグ**: `#水害リスク` `#水没` `#1階全喪失` `#孤立` `#避難困難`

---

### Case 2: 田主丸中央病院（河川氾濫事例）

**状況**: 福岡県久留米市の河川流域に位置

**結果**:
- 河川氾濫により1階が浸水
- 1階に設置されていた電気設備・揚水ポンプが水没
- 停電と断水が同時発生
- 病院機能が大幅に制限

**教訓**:
1. 電気設備（受変電設備、配電盤）は2階以上に設置
2. 揚水ポンプの電源は浸水しない場所から給電
3. 止水板、防水扉などの浸水対策を整備
4. 浸水時の電源遮断手順を明確化

**タグ**: `#水害リスク` `#電源水没` `#断水リスク` `#1階設備`

---

### Case 3: 水害対策が不明確な施設群

**対象施設例**: 済生会江津総合病院、市立福知山市民病院

**リスク**:
- 水害対策に関する記載がなく、備えの状況が不明
- 河川流域に位置する施設は浸水リスクが高い

**推奨対策**:
1. 自施設のハザードマップを確認
2. 浸水想定深さに応じた対策レベルを設定
3. 止水板・土嚢の備蓄
4. 排水ポンプの配備
5. 重要書類・データのバックアップ（上階またはクラウド）

**タグ**: `#水害対策不明` `#河川流域`
