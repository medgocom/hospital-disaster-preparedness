## 1. 優先ニーズランキング（質問1の結果）

### 📊 「自施設で取り組みたいこと」キーワードランキング

| 順位 | キーワード | 内容の要約 |
|:----:|------------|------------|
| 1位 | **アクションカード・マニュアルの整備** | 全スタッフが理解できるシンプルなマニュアル、部署ごとの役割分担の明確化 |
| 2位 | **災害BOX（防災BOX）の設置・活用** | 初動に必要な物品のパッケージ化、設置場所の周知 |
| 3位 | **通信・連絡手段の確保** | LINE等SNS活用、トランシーバー導入、衛星電話の操作習熟 |
| 4位 | **組織体制・発災基準（スイッチ）** | 災害対策本部の立ち上げ訓練、夜間・休日の組織図作成 |
| 5位 | **備蓄（食料・水）** | 職員・家族分の確保、備蓄場所の全職員への周知 |
| 6位 | **情報共有ツール（クロノロ・EMIS）** | クロノロジー訓練、EMIS入力・理解 |

### 📈 インフラ別 不安・関心度

| 関心度 | インフラ | 主な不安要素 | 取り組み要望 |
|:------:|----------|--------------|--------------|
| ⭐⭐⭐ 最高 | **通信** | 電話不通、連絡網なし | LINE連絡網、トランシーバー、衛星電話 |
| ⭐⭐⭐ 高い | **情報システム** | 電子カルテ停止、患者情報喪失 | 紙カルテ運用、患者マスター事前作成 |
| ⭐⭐⭐ 高い | **給食・食料** | 患者食提供、職員食料不足 | 非常食備蓄、保管場所周知 |
| ⭐⭐ 中程度 | **電気・燃料** | 停電時の機器対応 | 停電時マニュアル、非常用設備の限界時間把握 |
| ⭐ 低め | **水道・医療ガス** | （トイレ対策等で包括的に言及） | - |

### 💡 重要な発見

> **ハードウェアの整備よりも、発災直後の初動やインフラ途絶時の運用手順（ソフト面）への関心が圧倒的に高い**

## 2. 地域別特性分析（質問2の結果）

### 🏝️ 離島の施設（笠利、屋久島、徳之島）

**キーワード**: `津波` `備蓄` `避難` `孤立` `自己完結`

| 項目 | 内容 |
|------|------|
| 核心的不安 | **「逃げる・生き延びる」** - 救援が来るまでの数日間、限られた物資で自力維持 |
| 津波・避難 | 現在のマニュアルは「海に近づかない、高いところに逃げる」のみで具体性不足 |
| 物流途絶 | 非常食、災害BOX、生活用品（歯ブラシ、義歯洗浄等）の長期備蓄 |
| 特有の課題 | 病院避難基準の決定、災害対策委員会の立ち上げ |

### 🏙️ 都市部の施設（千葉、仙台、茅ヶ崎、湘南鎌倉）

**キーワード**: `通信` `参集` `トリアージ` `交通麻痺` `帰宅困難者`

| 項目 | 内容 |
|------|------|
| 核心的不安 | **「さばく・つなぐ」** - 膨大な情報と患者を処理する指揮系統維持 |
| 通信確保 | LINE活用、トランシーバー、衛星電話（電話回線パンク対策） |
| スタッフ参集 | 交通麻痺による出勤困難、夜間・休日の組織体制 |
| 特有の課題 | トリアージエリア設置、CSCA、クロノロジー、紙カルテ運用 |

### ❄️ 寒冷地の施設（札幌東、日高）

**キーワード**: `電源` `暖房` `燃料` `ブラックアウト` `凍死リスク`

| 項目 | 内容 |
|------|------|
| 核心的不安 | **「凍えない・止まらない」** - 電源喪失＝暖房停止＝生命危機 |
| 電源確保 | コージェネ不稼働、非常用コンセント使用不可の経験（2018年北海道胆振東部地震） |
| 冬期災害 | 雪害が高リスク、厳冬期の暖房確保は生命に直結 |
| 特有の課題 | 広域停電対策、積雪による交通網遮断、津波避難との両立 |

### 📋 地域別 重点課題サマリー

| 地域 | キーワード | 重点課題 |
|------|------------|----------|
| **離島** | 津波、備蓄、避難 | 数日間の自力維持、避難/放棄の判断基準 |
| **都市部** | 通信、参集、トリアージ | 指揮系統維持、交通麻痺下のスタッフ確保 |
| **寒冷地** | 電源、暖房、燃料 | ハード面の強靭化、燃料確保 |

## 3. トリアージ演習のボトルネック（質問3の結果）

### 🚧 現場職員が「動けない」「難しい」と感じる3つの課題

#### ボトルネック1: 動線（ゾーニング）の確保と物理的配置

**問題**: マニュアル上の知識はあっても、自施設の構造に当てはめた動線設計が困難

| 現場の声 |
|----------|
| 「出入口の明確化、動線」 |
| 「トリアージの動線の明確化」 |
| 「色分けや動線の効率性についてどうしたらスムーズになるのか学びたい」 |
| 「Ⅰ・Ⅱ・Ⅲのエリア区分やどこでトリアージすべきか」 |
| 「トリアージエリアの設置場所」 |

**対策案**:
- 自施設の図面を使った動線シミュレーション訓練
- エリア別の写真付きマニュアル作成
- 色分けテープ・サインの事前準備

---

#### ボトルネック2: 「判断に迷う症例」への対応

**問題**: START法を学んでも、典型例以外での判断に不安

| 現場の声 |
|----------|
| 「30秒で選択するために、何を一番にみるのか」 |
| 「START法で緑でも明らかに重症者である場合などどうするか」 |
| 「自分の感情をコントロールできない人（小さい子供や知的障害がある人）のトリアージ」 |
| 「実際の症例を見ての判断をどうするか」 |

**対策案**:
- 判断に迷う症例のケーススタディ集作成
- 「迷ったらこうする」ルールの明文化
- 実症例を使った模擬訓練の実施

---

#### ボトルネック3: 役割分担と継続的な運用の不明確さ

**問題**: 「誰が」行い、「どう連携し」、「時間経過でどう変わるか」が不明

| 現場の声 |
|----------|
| 「誰がトリアージするのか」 |
| 「PT（理学療法士）でもトリアージの現場でトリアージできるのか」 |
| 「トリアージ時の連携」 |
| 「2回目以降はどのような流れになるのか」 |
| 「トリアージは一度したら次回確認するのはどれくらい時間をあけた方がいいのか」 |

**対策案**:
- 職種別の役割分担表作成
- 再トリアージのタイミング・ルール明文化
- 時間経過に伴う組織体制の変化図
